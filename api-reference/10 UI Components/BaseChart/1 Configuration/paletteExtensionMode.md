##### shortDescription
Specifies what to do with colors in the [palette](/api-reference/20%20Data%20Visualization%20Widgets/BaseChart/1%20Configuration/palette.md '{basewidgetpath}/Configuration/#palette') when their number is less than the number of series (in the **Chart** UI component) or points in a series (in the **PieChart** UI component).
