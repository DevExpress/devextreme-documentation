##### shortDescription
Provides information about the selection state of a point.

##### return: Boolean
Selection state of a point.

---
When a point is selected, this method returns **true**. Otherwise, the returned value is **false**.

You can select a point using its **select()** method. To learn more about point and series selection, refer to the [Selection Handling](/concepts/05%20Widgets/zz%20Common/10%20Data%20Visualization%20Widgets/90%20Charts%20-%20End-User%20Interaction/4%20Selection%20Handling '/Documentation/Guide/Widgets/Common/Data_Visualization_Widgets/Charts_-_End-User_Interaction/Selection_Handling') topic.
