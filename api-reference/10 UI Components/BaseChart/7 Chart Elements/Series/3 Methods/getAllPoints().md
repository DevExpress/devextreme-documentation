##### shortDescription
Gets all points in the series.

##### return: Array<basePointObject>
All points in the series.

---
[note] A series point is a data object's visual representation. It can be selected, hovered over, or hidden. You can access the data object that the series point represents using the point's [data](/api-reference/20%20Data%20Visualization%20Widgets/BaseChart/7%20Chart%20Elements/Point/2%20Fields/data.md '{basewidgetpath}/Chart_Elements/Point/Fields/#data') field.
