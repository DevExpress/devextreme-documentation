##### shortDescription
This section describes the **Series** object, which represents a series.

---
When you get a series as the return value of a function, or a callback function parameter, you can use the members of the **Series** object.

To get the required series, use the chart's functions (e.g., [getAllSeries()](/api-reference/20%20Data%20Visualization%20Widgets/BaseChart/3%20Methods/getAllSeries().md '/Documentation/ApiReference/UI_Components/dxChart/Methods/#getAllSeries'), [getSeriesByName(seriesName)](/api-reference/20%20Data%20Visualization%20Widgets/BaseChart/3%20Methods/getSeriesByName(seriesName).md '/Documentation/ApiReference/UI_Components/dxChart/Methods/#getSeriesByNameseriesName')).
