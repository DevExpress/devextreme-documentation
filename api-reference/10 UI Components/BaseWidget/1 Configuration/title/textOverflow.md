##### shortDescription
Specifies what to do with the title when it overflows the allocated space after applying [wordWrap](/api-reference/20%20Data%20Visualization%20Widgets/BaseWidget/1%20Configuration/title/wordWrap.md '{basewidgetpath}/Configuration/title/#wordWrap'): hide, truncate it and display an ellipsis, or do nothing.
