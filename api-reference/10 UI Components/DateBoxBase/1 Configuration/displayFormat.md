##### shortDescription
Specifies the date display format. Ignored if the [pickerType]({basewidgetpath}/Configuration/#pickerType) property is *"native"*

---
Set [useMaskBehavior]({basewidgetpath}/Configuration/#useMaskBehavior) to **true** to control user input according to the display format.

The default display format is selected automatically based on the current [locale](/Documentation/ApiReference/Common/utils/localization/#locale).
