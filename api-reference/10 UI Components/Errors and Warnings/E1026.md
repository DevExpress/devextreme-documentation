##### shortDescription
Occurs in the [DataGrid](/api-reference/10%20UI%20Components/dxDataGrid '/Documentation/ApiReference/UI_Components/dxDataGrid/') UI component if the **summaryType** field of an item is set to "custom", provided that the [calculateCustomSummary](/api-reference/10%20UI%20Components/dxDataGrid '/Documentation/ApiReference/UI_Components/dxDataGrid/'Configuration/summary/#calculateCustomSummary) property value is not defined.

---
Define the **calculateCustomSummary** property value.
