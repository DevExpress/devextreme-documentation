##### shortDescription
Occurs when the component data source contains items with non-unique key values.

---
This error can occur in the following components:

- [Accordion](/Documentation/ApiReference/UI_Components/dxAccordion/)
- [List](/Documentation/ApiReference/UI_Components/dxList/)
- [TabPanel](/Documentation/ApiReference/UI_Components/dxTabPanel/)
- [Tabs](/Documentation/ApiReference/UI_Components/dxTabs/)
- [TreeView](/Documentation/ApiReference/UI_Components/dxTreeView/)

To resolve this error, ensure all items have unique key values.
