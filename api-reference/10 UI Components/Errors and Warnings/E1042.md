##### shortDescription
Occurs in the **DataGrid** or **TreeList** UI component if the key field is not set.

---
If your data is a [JavaScript array](/concepts/70%20Data%20Binding/00%20Specify%20a%20Data%20Source/10%20Local%20Array.md '/Documentation/Guide/Data_Binding/Specify_a_Data_Source/Local_Array/'), use the UI component's [keyExpr](/api-reference/10%20UI%20Widgets/dxDataGrid/1%20Configuration/keyExpr.md '/Documentation/ApiReference/UI_Widgets/dxDataGrid/Configuration/#keyExpr') option to specify the key field.  Otherwise, use the **key** option in the store.
