##### shortDescription
Occurs when the **key** option of the [Store](/concepts/70%20Data%20Binding/5%20Data%20Layer/1%20Creating%20DataSource/3%20What%20Are%20Stores.md '/Documentation/Guide/Data_Binding/Data_Layer/#Creating_DataSource/What_Are_Stores') has an incorrect value.

---
Make sure that the field specified in the **key** option exists in all data objects associated with the **Store**.
