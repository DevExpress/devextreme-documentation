##### shortDescription
Occurs in a text editor when at least one of its [buttons](/api-reference/10%20UI%20Widgets/dxTextEditor/1%20Configuration/buttons '/Documentation/ApiReference/UI_Widgets/dxTextBox/Configuration/buttons/') does not have a [name](/api-reference/_hidden/dxTextEditorButton/name.md '/Documentation/ApiReference/UI_Widgets/dxTextBox/Configuration/buttons/#name').

---
Check that the **name** option is specified for all the buttons.
