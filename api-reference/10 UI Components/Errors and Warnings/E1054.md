##### shortDescription
Occurs in a text editor when at least one of its [buttons](/api-reference/10%20UI%20Components/dxTextEditor/1%20Configuration/buttons '/Documentation/ApiReference/UI_Components/dxTextBox/Configuration/buttons/') does not have a [name](/api-reference/10%20UI%20Components/dxTextEditor/1%20Configuration/buttons '/Documentation/ApiReference/UI_Components/dxTextBox/Configuration/buttons/'#name).

---
Check that the **name** property is specified for all the buttons.
