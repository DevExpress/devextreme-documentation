##### shortDescription
Occurs in the [SpeechToText](/Documentation/ApiReference/UI_Components/dxSpeechToText/) component when the browser does not support the [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API).

---
Certain browsers such as Mozilla Firefox do not support the Web Speech API [SpeechRecognition](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API#api.SpeechRecognition) functionality. To resolve this error, implement a custom speech recognition engine through the [customSpeechRecognizer](/Documentation/ApiReference/UI_Components/dxSpeechToText/Configuration/customSpeechRecognizer/) property.
