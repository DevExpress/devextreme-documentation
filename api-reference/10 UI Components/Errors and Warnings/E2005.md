##### shortDescription
Occurs in the Funnel UI component when the specified value field is absent in the data source or all its values are negative.

---
Check that you set the [valueField](/api-reference/10%20UI%20Components/dxFunnel/1%20Configuration/valueField.md '/Documentation/ApiReference/UI_Components/dxFunnel/Configuration/#valueField') property to a field in the data source containing at least one positive value.
