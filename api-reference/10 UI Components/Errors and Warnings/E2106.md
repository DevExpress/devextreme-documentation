---
##### shortDescription
Occurs when the visible range specified by the [min](/api-reference/10%20UI%20Components/dxChart/1%20Configuration/argumentAxis/min.md '/Documentation/ApiReference/UI_Components/dxChart/Configuration/argumentAxis/#min') and [max](/api-reference/10%20UI%20Components/dxChart/1%20Configuration/argumentAxis/max.md '/Documentation/ApiReference/UI_Components/dxChart/Configuration/argumentAxis/#max') properties of a chart axis is not valid.

---
The **min** and **max** properties accept a numeric or date-time value depending on the [data type](/api-reference/10%20UI%20Components/dxChart/1%20Configuration/argumentAxis/argumentType.md '/Documentation/ApiReference/UI_Components/dxChart/Configuration/argumentAxis/#argumentType'). If anything else is passed, the E2106 error occurs.
