##### shortDescription
Occurs in the [Scheduler](/api-reference/10%20UI%20Components/dxScheduler '/Documentation/ApiReference/UI_Components/dxScheduler/') UI component if you try to scroll to a date outside the current view.

---
Check that the [scrollTo(date, group, allDay)](/Documentation/ApiReference/UI_Components/dxScheduler/Methods/#scrollTodate_group_allDay) method is called with a date in the current view.
