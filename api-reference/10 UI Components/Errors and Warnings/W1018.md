##### shortDescription
Occurs in the DataGrid UI component when infinite scrolling is used with multiple selection.

---
Set **selection**.[deferred](/Documentation/ApiReference/UI_Components/dxDataGrid/Configuration/selection/#deferred) to **true** or set **selection**.[selectAllMode](/Documentation/ApiReference/UI_Components/dxDataGrid/Configuration/selection/#selectAllMode) to *"page"*.
