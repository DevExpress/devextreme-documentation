##### shortDescription
Occurs in the Scheduler when the [hiddenWeekDays](/Documentation/ApiReference/UI_Components/dxScheduler/Configuration/#hiddenWeekDays) property specifies all seven days of the week.

---
The Scheduler ignores the **hiddenWeekDays** value and keeps all days visible. Remove at least one day from the array so that at least one day remains visible.
