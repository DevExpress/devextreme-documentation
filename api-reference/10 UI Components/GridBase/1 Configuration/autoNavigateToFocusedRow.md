---
id: GridBase.Options.autoNavigateToFocusedRow
type: Boolean
default: true
---
---
##### shortDescription
Automatically scrolls the component to the focused row when the [focusedRowKey](/api-reference/10%20UI%20Components/GridBase/1%20Configuration/focusedRowKey.md '/Documentation/ApiReference/UI_Components/dxDataGrid/Configuration/#focusedRowKey') is changed.

---

[note] You must specify the component's [height](/api-reference/10%20UI%20Components/DOMComponent/1%20Configuration/height.md '/Documentation/ApiReference/UI_Components/dx{WidgetName}/Configuration/#height') to ensure that the **autoNavigateToFocusedRow** property works properly.
