##### shortDescription
Configures the column chooser.

---
The column chooser allows a user to hide columns at runtime. To enable it, assign **true** to the **columnChooser**.[enabled](/api-reference/40%20Common%20Types/ColumnChooser/enabled.md '{basewidgetpath}/Configuration/columnChooser/#enabled') property.
