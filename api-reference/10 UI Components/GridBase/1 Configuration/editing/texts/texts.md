##### shortDescription
Contains options that specify texts for editing-related UI elements.

---
The following code shows the **editing**.**texts** declaration syntax:

---
##### jQuery

    <!-- tab: index.js -->
    $(function() {
        $("#{widgetName}Container").dx{WidgetName}({
            // ...
            editing: {
                // ...
                texts: {
                    deleteRow: "Remove"
                }
            }
        });
    });
