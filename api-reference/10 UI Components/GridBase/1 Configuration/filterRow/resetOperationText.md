##### shortDescription
Specifies text for the reset operation on the filter list.

---
The reset operation restores the default filter, which depends on the column's [data type](/api-reference/_hidden/GridBaseColumn/dataType.md '{basewidgetpath}/Configuration/columns/#dataType'). For details on default filters by data types, refer to the description of the [selectedFilterOperation](/api-reference/_hidden/GridBaseColumn/selectedFilterOperation.md '{basewidgetpath}/Configuration/columns/#selectedFilterOperation') option.
