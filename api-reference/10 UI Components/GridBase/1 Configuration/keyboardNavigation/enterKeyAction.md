##### shortDescription
Specifies whether the Enter key switches the cell or row to the edit state or moves focus in the [enterKeyDirection](/api-reference/10%20UI%20Components/GridBase/1%20Configuration/keyboardNavigation/enterKeyDirection.md '{basewidgetpath}/Configuration/keyboardNavigation/#enterKeyDirection'). Applies for all [edit modes](/api-reference/10%20UI%20Widgets/GridBase/1%20Configuration/editing/mode.md '{basewidgetpath}/Configuration/editing/#mode'), except *"popup"*.
