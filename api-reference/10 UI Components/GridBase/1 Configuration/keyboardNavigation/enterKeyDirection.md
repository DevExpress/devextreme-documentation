##### shortDescription
Specifies the direction in which to move focus when a user presses Enter. Applies if **editing**.[mode](/api-reference/10%20UI%20Widgets/GridBase/1%20Configuration/editing/mode.md '{basewidgetpath}/Configuration/editing/#mode') is *"cell"* or *"batch"*.
