##### shortDescription
Specifies the page size.

---
[note] The page size determines how many rows the UI component loads at a time. All the loaded rows are rendered. If this reduces the UI component's speed, set the [rowRenderingMode](/api-reference/10%20UI%20Widgets/GridBase/1%20Configuration/scrolling/rowRenderingMode.md '{basewidgetpath}/Configuration/scrolling/#rowRenderingMode') to "virtual" to render only the rows in the viewport. Alternatively, you can decrease the page size.
