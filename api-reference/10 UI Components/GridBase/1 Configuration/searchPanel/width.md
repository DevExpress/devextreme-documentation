---
##### shortDescription
Specifies the width of the search panel.

---
This property accepts a value of one of the following types:

- **Number**  
The width in pixels.

- **String**  
A CSS-accepted measurement of width. For example, `"55px"`, `"20vw"`, `"80%"`, `"auto"`, `"inherit"`.
