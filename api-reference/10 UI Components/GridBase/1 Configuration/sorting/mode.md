##### shortDescription
Specifies the sorting mode.

---
The following sorting modes are available in the UI component:

- **Single**        
Rows can be sorted by values of a single column only.

- **Multiple**      
Rows can be sorted by values of several columns.

- **None**     
Sorting is disabled.
