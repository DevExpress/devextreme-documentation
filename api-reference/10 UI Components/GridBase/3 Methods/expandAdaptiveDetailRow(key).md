##### shortDescription
Expands an adaptive detail row.

##### param(key): any
The key of the data row to which the adaptive detail row belongs.

---
To access a data row by its key, you should specify the field that provides keys in the [data source](/api-reference/10%20UI%20Widgets/GridBase/1%20Configuration/dataSource.md '{basewidgetpath}/Configuration/#dataSource'). If no key was specified, the whole data object is considered the key.
