##### shortDescription
Collapses an item with a specific index.

##### return: Promise<void>
A Promise that is resolved if the item is collapsed and rejected otherwise.
#include ref-promisedistinction

##### param(index): Number
The item's index.

---
[note]The method applies when the [collapsible](/api-reference/10%20UI%20Components/dxAccordion/1%20Configuration/collapsible.md '/Documentation/ApiReference/UI_Components/dxAccordion/Configuration/#collapsible') property is set to **true**.
