---
id: dxActionSheet.Options.showCancelButton
type: Boolean
default: true
---
---
##### shortDescription
Specifies whether or not to display the **Cancel** button in action sheet.

---
The **Cancel** button closes the action sheet.

[note]The **showCancelButton** property makes sense only if the [usePopover](/api-reference/10%20UI%20Widgets/dxActionSheet/1%20Configuration/usePopover.md '/Documentation/ApiReference/UI_Components/dxActionSheet/Configuration/#usePopover') property is disabled. Otherwise, the **Cancel** button is hidden. In this case, the popover is closed by an outside click.
