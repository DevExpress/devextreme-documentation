##### shortDescription
Contains the keys of the selected buttons and allows selecting buttons initially.

---
The data field that provides the keys can be specified using the [keyExpr](/api-reference/10%20UI%20Widgets/dxButtonGroup/1%20Configuration/keyExpr.md '/Documentation/ApiReference/UI_Widgets/dxButtonGroup/Configuration/#keyExpr') option.
