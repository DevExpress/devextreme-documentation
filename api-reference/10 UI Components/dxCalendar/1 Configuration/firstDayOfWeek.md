##### shortDescription
Specifies the first day of a week.

---
The option can take on a value from 0 to 6.

- 0 - Sunday
- 1 - Monday
- 2 - Tuesday
- 3 - Wednesday
- 4 - Thursday
- 5 - Friday
- 6 - Saturday

By default, the value provided by the culture settings is used.
