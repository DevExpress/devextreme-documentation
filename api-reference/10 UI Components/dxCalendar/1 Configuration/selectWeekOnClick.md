---
id: dxCalendar.Options.selectWeekOnClick
type: Boolean
default: true
---
---
##### shortDescription
Specifies whether a user can select a week by clicking on a week number.

---
This property is in effect if [showWeekNumbers](/api-reference/10%20UI%20Components/dxCalendar/1%20Configuration/showWeekNumbers.md '/Documentation/ApiReference/UI_Components/dxCalendar/Configuration/#showWeekNumbers') is enabled and [selectionMode](/api-reference/10%20UI%20Components/dxCalendar/1%20Configuration/selectionMode.md '/Documentation/ApiReference/UI_Components/dxCalendar/Configuration/#selectionMode') is *'multiple'* or *'range'*.
