##### shortDescription
Specifies whether to synchronize the [header filter](/api-reference/10%20UI%20Components/dxCardView/1%20Configuration/headerFilter.md '/Documentation/ApiReference/UI_Components/dxCardView/Configuration/#headerFilter') and [filter builder](/api-reference/10%20UI%20Components/dxCardView/1%20Configuration/filterBuilder.md '/Documentation/ApiReference/UI_Components/dxCardView/Configuration/#filterBuilder'). The synchronized filter expression is stored in the [filterValue](/api-reference/10%20UI%20Components/dxCardView/1%20Configuration/filterValue.md '{basewidgetpath}/Configuration/#filterValue') property.
