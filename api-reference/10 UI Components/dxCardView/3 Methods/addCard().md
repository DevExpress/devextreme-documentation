##### shortDescription
Adds an empty card and switches it to the editing state.

---
Use this method if you want to add an empty card.

This method operates when **paging**.[enabled](/api-reference/10%20UI%20Components/dxCardView/9%20Types/Paging/enabled.md '{basewidgetpath}/Configuration/paging/#enabled') is set to `false`, or when **dataSource**.[reshapeOnPush](/api-reference/30%20Data%20Layer/DataSource/1%20Configuration/reshapeOnPush.md '/Documentation/ApiReference/Data_Layer/DataSource/Configuration/#reshapeOnPush') is `true` and [remoteOperations](/api-reference/10%20UI%20Components/dxCardView/9%20Types/RemoteOperations '{basewidgetpath}/Configuration/remoteOperations/') is `false`.
