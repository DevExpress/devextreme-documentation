##### shortDescription
Adds an empty card and switches it to the editing state.

---
Use this method if you want to add an empty card.

This method operates when **paging**.[enabled]({basewidgetpath}/Configuration/paging/#enabled) is set to `false`, or when **dataSource**.[reshapeOnPush](/api-reference/30%20Data%20Layer/DataSource/1%20Configuration/reshapeOnPush.md '/Documentation/ApiReference/Data_Layer/DataSource/Configuration/#reshapeOnPush') is `true` and [remoteOperations]({basewidgetpath}/Configuration/remoteOperations/) is `false`.
