##### shortDescription
Clears the selection of all cards on all pages or the currently rendered page only.

---
The value of the [selectAllMode]({basewidgetpath}/Configuration/selection/#selectAllMode) property determines whether this method clears selections from all cards across all pages or only from visible pages. If a filter is in effect, selections clear only from cards that match filter conditions. To clear all selections regardless **selectAllMode** or filters, use the [clearSelection()]({basewidgetpath}/Methods/#clearSelection) method.
