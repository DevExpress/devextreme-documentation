##### shortDescription
Hides the [load panel]({basewidgetpath}/Configuration/loadPanel/).

---
CardView hides the load panel automatically when data is ready. If you used the [beginCustomLoading(text)]({basewidgetpath}/Methods/#beginCustomLoadingtext) method to display the load panel, call the **endCustomLoading()** method to remove it.
