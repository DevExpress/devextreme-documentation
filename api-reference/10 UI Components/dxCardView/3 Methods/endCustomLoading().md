##### shortDescription
Hides the [load panel](/api-reference/10%20UI%20Components/dxCardView/1%20Configuration/loadPanel.md '{basewidgetpath}/Configuration/loadPanel/').

---
CardView hides the load panel automatically when data is ready. If you used the [beginCustomLoading(text)](/api-reference/10%20UI%20Components/dxCardView/3%20Methods/beginCustomLoading(text).md '{basewidgetpath}/Methods/#beginCustomLoadingtext') method to display the load panel, call the **endCustomLoading()** method to remove it.
