---
##### shortDescription
Gets the selected card keys.

##### return: Array<any>
Keys of selected cards. The keys are stored in the order the user selects cards.

---
The whole data object becomes a key if no field is specified in the data source for keys. This method then retrieves data objects similar to the [getSelectedCardsData()]({basewidgetpath}/Methods/#getSelectedCardsData).
