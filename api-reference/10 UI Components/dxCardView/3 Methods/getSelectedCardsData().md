##### shortDescription
Gets the selected cards' data objects.

##### return: Array<any>
The selected cards' data objects.        
The objects are not processed by the [DataSource](/api-reference/30%20Data%20Layer/DataSource '/Documentation/ApiReference/Data_Layer/DataSource/') and have the same order in which the cards were selected.
