##### shortDescription
Searches for a string in columns whose [allowSearch](/api-reference/10%20UI%20Components/dxCardView/9%20Types/ColumnProperties/allowSearch.md '{basewidgetpath}/Configuration/columns/#allowSearch') property is `true`.

##### param(text): String
A search string. Pass an empty string to clear search results.
