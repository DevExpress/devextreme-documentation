##### shortDescription
Specifies whether this column can be searched. Applies only if [searchPanel](/api-reference/10%20UI%20Components/dxCardView/1%20Configuration/searchPanel.md '{basewidgetpath}/Configuration/searchPanel/').**visible** is `true`. Inherits the value of the [allowFiltering](/api-reference/10%20UI%20Components/dxCardView/9%20Types/ColumnProperties/allowFiltering.md '{basewidgetpath}/Configuration/columns/#allowFiltering') property.
