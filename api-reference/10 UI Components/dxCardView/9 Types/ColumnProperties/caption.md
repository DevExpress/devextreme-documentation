##### shortDescription
Specifies a caption for the column.

---
Use this property to display a descriptive or friendly name for the column. If this property is not set, the caption will be generated from the name of the [dataField](/api-reference/10%20UI%20Components/dxCardView/9%20Types/ColumnProperties/dataField.md '{basewidgetpath}/Configuration/columns/#dataField').
