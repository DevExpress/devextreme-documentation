##### shortDescription
Configures the form item that the field produces in the editing state.

---
[note] The component does not check validation rules in the **formItem** object. Use the **columns**.[validationRules](/api-reference/10%20UI%20Components/dxCardView/9%20Types/ColumnProperties/validationRules.md '{basewidgetpath}/Configuration/columns/#validationRules') property to customize validation instead.
