##### shortDescription
Configures paging.

---
Paging allows the UI component to load data in portions instead of loading it simultaneously. To enable paging, set the **paging**.[enabled]({basewidgetpath}/Configuration/paging/#enabled) property to `true`.

Users can switch between pages and change paging settings using the [pager]({basewidgetpath}/Configuration/pager/) or they can scroll the pages. Paging settings apply with any scrolling mode.
