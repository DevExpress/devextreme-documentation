##### shortDescription
Configures paging.

---
Paging allows the UI component to load data in portions instead of loading it simultaneously. To enable paging, set the **paging**.[enabled](/api-reference/10%20UI%20Components/dxCardView/9%20Types/Paging/enabled.md '{basewidgetpath}/Configuration/paging/#enabled') property to `true`.

Users can switch between pages and change paging settings using the [pager](/api-reference/10%20UI%20Components/dxCardView/1%20Configuration/pager.md '{basewidgetpath}/Configuration/pager/') or they can scroll the pages. Paging settings apply with any scrolling mode.
