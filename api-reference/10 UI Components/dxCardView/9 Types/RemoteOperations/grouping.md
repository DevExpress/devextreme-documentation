##### shortDescription
Specifies whether grouping must be performed on the server.

---
Grouping is required only when a user can filter data using a [header filter](/api-reference/10%20UI%20Components/dxCardView/1%20Configuration/headerFilter.md '{basewidgetpath}/Configuration/headerFilter/').
