##### shortDescription
Configures runtime selection.

---
A user can select rows in a single or multiple [mode](/api-reference/10%20UI%20Components/dxCardView/9%20Types/SelectionConfiguration/mode.md '{basewidgetpath}/Configuration/selection/#mode'). In multiple mode, a user can select all rows at once. To disable this feature, assign `false` to the [allowSelectAll](/api-reference/10%20UI%20Components/dxCardView/9%20Types/SelectionConfiguration/allowSelectAll.md '{basewidgetpath}/Configuration/selection/#allowSelectAll') property.
