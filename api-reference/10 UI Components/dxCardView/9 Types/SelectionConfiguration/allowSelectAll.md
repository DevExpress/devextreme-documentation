---
##### shortDescription
Allows users to simultaneously select all or current page cards (depending on [selectAllMode](/api-reference/10%20UI%20Components/dxCardView/9%20Types/SelectionConfiguration/selectAllMode.md '{basewidgetpath}/Configuration/selection/#selectAllMode')).

---
Press Ctrl/Cmd + A or click the "Select all" button above the header panel to select all cards. To remove all selections, click the "Clear selection" button. When a filter is active, selecting all cards affects those matching the filter.

When set to `false`, this property turns off the "Select all" functionality.
