##### shortDescription
Configures the argument axis.

---
![DevExtreme HTML5 Charts Axis Axes](/images/ChartJS/visual_elements/axes.png)

The **argumentAxis** object, which is described here, configures the argument axis individually. To specify common settings for all axes in a chart, use the [commonAxisSettings](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/commonAxisSettings '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonAxisSettings/') object. Axis-specific settings override common settings.
