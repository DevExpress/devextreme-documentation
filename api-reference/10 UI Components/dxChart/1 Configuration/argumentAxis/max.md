##### shortDescription
Coupled with the **min** option, focuses the UI component on a specific chart segment. Applies only to the axes of the *"continuous"* and *"logarithmic"* [type](/api-reference/10%20UI%20Components/dxChart/1%20Configuration/argumentAxis/type.md '/Documentation/ApiReference/UI_Components/dxChart/Configuration/argumentAxis/#type').
