##### shortDescription
Coupled with the **max** property, focuses the UI component on a specific chart segment. Applies only to the axes of the *"continuous"* and *"logarithmic"* [type](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/argumentAxis/type.md '/Documentation/ApiReference/UI_Components/dxChart/Configuration/argumentAxis/#type').
