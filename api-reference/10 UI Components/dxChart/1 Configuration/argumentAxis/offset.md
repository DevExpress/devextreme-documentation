##### shortDescription
Specifies the shift in pixels of the argument axis.

---
Negative numbers shift the axis position to the top, positive numbers - to the bottom. If the [rotated](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/rotated.md '/Documentation/ApiReference/UI_Components/dxChart/Configuration/#rotated') property is enabled, negative numbers will shift the axis position to the left, and positive numbers - to the right.
