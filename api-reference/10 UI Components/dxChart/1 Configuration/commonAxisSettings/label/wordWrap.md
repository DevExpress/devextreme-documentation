default: 'normal'

---
##### shortDescription
Specifies how to wrap texts that do not fit into a single line.

---
This property takes effect for the vertical axis only when space for labels is limited. Use the [placeholderSize](/Documentation/ApiReference/UI_Components/dxChart/Configuration/valueAxis/#placeholderSize) property to specify the size of the label panel.
