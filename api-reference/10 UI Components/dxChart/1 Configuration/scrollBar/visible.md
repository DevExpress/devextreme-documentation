---
id: dxChart.Options.scrollBar.visible
type: Boolean
default: false
---
---
##### shortDescription
Specifies whether the scroll bar is visible or not.

---
[note]Making the scroll bar visible can be useful only if [panning is enabled](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/zoomAndPan '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/zoomAndPan/') in your chart.
