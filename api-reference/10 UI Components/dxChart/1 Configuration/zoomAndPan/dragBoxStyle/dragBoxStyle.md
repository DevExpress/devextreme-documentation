##### shortDescription
Configures the box that appears when users zoom the chart by selecting an area with the drag gesture. Applies only if [dragToZoom](/api-reference/10%20UI%20Components/dxChart/1%20Configuration/zoomAndPan/dragToZoom.md '/Documentation/ApiReference/UI_Components/dxChart/Configuration/zoomAndPan/#dragToZoom') is **true**.
