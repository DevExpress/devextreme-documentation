##### shortDescription
Specifies the key that enables panning when [dragToZoom](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/zoomAndPan/dragToZoom.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/zoomAndPan/#dragToZoom') is **true**. Applies only on devices that use a mouse.

---
[note] The Meta key depends on the keyboard: the Command key (&#8984;) on Macintosh keyboards; the Windows key (&#8862;) on Windows keyboards.
