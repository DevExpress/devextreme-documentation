##### shortDescription
Configures series points in scatter, line- and area-like series.

Declared in [commonSeriesSettings](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/commonSeriesSettings '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonSeriesSettings/'), the **point** settings apply to all points in the chart. Declared in a [series](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/series '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/series/') configuration object, the **point** settings apply only to the points that belong to this particular series. The series-specific **point** settings override the common ones.
