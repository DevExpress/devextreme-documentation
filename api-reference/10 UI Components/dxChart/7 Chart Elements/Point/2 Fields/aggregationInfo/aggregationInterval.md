##### shortDescription
Contains the length of the aggregation interval in axis units (numbers or dates). If the interval is set in pixels (using the [aggregationGroupWidth](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/argumentAxis/aggregationGroupWidth.md '/Documentation/ApiReference/UI_Components/dxChart/Configuration/argumentAxis/#aggregationGroupWidth') option), it will be converted to axis units.
