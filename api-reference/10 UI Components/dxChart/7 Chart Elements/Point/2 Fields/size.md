##### shortDescription
Contains the size of the bubble as it was set in the data source. This field is useful for points belonging to a series of the [bubble](/api-reference/10%20UI%20Components/dxChart/5%20Series%20Types/BubbleSeries '/Documentation/ApiReference/UI_Components/dxChart/Series_Types/BubbleSeries/') type only.
