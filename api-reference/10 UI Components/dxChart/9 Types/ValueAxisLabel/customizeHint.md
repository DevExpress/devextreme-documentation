##### shortDescription
Specifies the hint that appears when a user points to an axis label.

##### param(axisValue): Object
Information on the axis value.

##### field(axisValue.value): Date | Number | String
The raw value.

##### field(axisValue.valueText): String
The [formatted](/api-reference/10%20UI%20Components/dxChart/1%20Configuration/valueAxis/label/format.md '/Documentation/ApiReference/UI_Components/dxChart/Configuration/valueAxis/label/#format') value converted to a string.

##### return: String
The text for the hint to display.
