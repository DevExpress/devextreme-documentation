uid: viz\chart:dxChartArgumentAxisConstantLines.extendAxis
type: Boolean
default: false

---
##### shortDescription
Specifies whether to extend the axis's default [visual range](/api-reference/10%20UI%20Components/dxChart/1%20Configuration/argumentAxis/visualRange '/Documentation/ApiReference/UI_Components/dxChart/Configuration/argumentAxis/visualRange/') to display the constant line.

---
<!--
The default visual range equals the data range. A constant line outside the visual range is not displayed. Set this property to **true** to extend the range and display the constant line.
-->
