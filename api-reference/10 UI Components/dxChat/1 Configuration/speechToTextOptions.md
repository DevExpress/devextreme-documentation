##### shortDescription
Configures Speech-to-Text options.

---
To enable the Speech-to-Text feature, set the [speechToTextEnabled](/Documentation/ApiReference/UI_Components/dxChat/Configuration/#speechToTextEnabled) option to `true`. Use the **speechToTextOptions** property to configure the feature's behavior and appearance or to specify a [customSpeechRecognizer](/Documentation/ApiReference/UI_Components/dxSpeechToText/Configuration/#customSpeechRecognizer).

You can specify most [SpeechToText properties](/Documentation/ApiReference/UI_Components/dxSpeechToText/Configuration/) in this object, except for the properties listed below. The Chat component overrides these properties.

- [stylingMode](/Documentation/ApiReference/UI_Components/dxSpeechToText/Configuration/#stylingMode)
- [type](/Documentation/ApiReference/UI_Components/dxSpeechToText/Configuration/#type)
