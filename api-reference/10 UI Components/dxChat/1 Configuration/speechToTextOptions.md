##### shortDescription
Configures Speech-to-Text options.

---
To enable the Speech-to-Text feature, set the [speechToTextEnabled](/Documentation/ApiReference/UI_Components/dxChat/Configuration/#speechToTextEnabled) option to `true`. Use this option to configure the feature's behavior and appearance or to specify [customSpeechRecognizer](/Documentation/ApiReference/UI_Components/dxSpeechToText/Configuration/#customSpeechRecognizer)ю Use the [inputFieldText](/Documentation/ApiReference/UI_Components/dxChat/Configuration/#inputFieldText) option to get the dictated text.

You can specify most [SpeechToText properties](/Documentation/ApiReference/UI_Components/dxSpeechToText/Configuration/) in this object, except for the properties listed below. Chat overrides these properties.

- [stylingMode](/Documentation/ApiReference/UI_Components/dxSpeechToText/Configuration/#stylingMode)
- [type](/Documentation/ApiReference/UI_Components/dxSpeechToText/Configuration/#type)
