##### shortDescription
Image alternative text. Specifies the value of image `alt` attributes. Screen readers utilize this property when image messages are focused. Only applies to messages of [type](/Documentation/ApiReference/UI_Components/dxChat/Types/Message/#type) `image`.

---
This property has a default placeholder value. To improve your project's accessibility, define a descriptive alt text.
