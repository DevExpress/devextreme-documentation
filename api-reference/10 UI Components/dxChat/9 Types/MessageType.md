##### shortDescription
Specifies the chat message type.

---
Use this type to define the format of a message in the [type](/api-reference/10%20UI%20Components/dxChat/9%20Types/MessageBase/type.md '/Documentation/ApiReference/UI_Components/dxChat/Types/Message/#type') field.

The following values are accepted:

- `'text'` - A plain-text message.
- `'image'` - A message containing an image.
