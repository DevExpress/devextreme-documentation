##### shortDescription
Specifies the UI component state.

---
The check box can be in one of the following states depending on the **value** property's value.

- checked  
 **value** is **true**

- unchecked  
 **value** is **false**

- undetermined  
 **value** is **undefined** or **null**

Users can press the Space key to change the **value**.
