##### shortDescription
Specifies the UI component state.

---
The check box can be in one of the following states depending on the **value** property value.

- checked  
 value is **true**

- unchecked  
 value is **false**

- undetermined  
 value is **undefined**

An end-user can change the UI component value using the **Space** key.
