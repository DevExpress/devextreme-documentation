##### shortDescription
Specifies whether groups appear expanded or not.

---
[note] When you use the [ODataStore](/api-reference/30%20Data%20Layer/ODataStore '/Documentation/ApiReference/Data_Layer/ODataStore/') and set this option to **false**, the **DataGrid** loads data for all pages at once.
