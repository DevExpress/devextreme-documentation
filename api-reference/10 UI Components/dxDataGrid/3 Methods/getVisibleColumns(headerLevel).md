##### shortDescription
Gets all visible columns at a specific hierarchical level of column headers. Use it to access [banded columns](/concepts/05%20UI%20Components/DataGrid/15%20Columns/10%20Column%20Types/2%20Band%20Columns.md '/Documentation/Guide/UI_Components/DataGrid/Columns/Column_Types/Band_Columns/').

##### return: Array<dxDataGridColumn>
Visible columns; may include command columns.

##### param(headerLevel): Number
The column headers' level.
