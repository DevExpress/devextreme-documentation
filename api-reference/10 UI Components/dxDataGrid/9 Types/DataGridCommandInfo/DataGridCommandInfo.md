##### shortDescription
Information about a predefined DataGrid command.

---
**DataGridCommandInfo** contains the following fields:

- **name**: The predefined command's name ([DataGridPredefinedCommandNames](/api-reference/10%20UI%20Components/dxDataGrid/9%20Types/DataGridPredefinedCommandNames '{basewidgetpath}/Types/DataGridPredefinedCommandNames/')).
- **args**: Command arguments. Refer to [DataGridPredefinedCommands](/api-reference/10%20UI%20Components/dxDataGrid/9%20Types/DataGridPredefinedCommands '{basewidgetpath}/Types/DataGridPredefinedCommands/') for information about the arguments of each available command.
