---
id: dxDateBox.Options.interval
type: Number
default: 30
---
---
##### shortDescription
Specifies the interval between neighboring values in the popup list in minutes.

---
[note]This option applies only if the [type](/api-reference/10%20UI%20Widgets/dxDateBox/1%20Configuration/type.md '/Documentation/ApiReference/UI_Widgets/dxDateBox/Configuration/#type') option is set to *"time"* and the [pickerType](/api-reference/10%20UI%20Widgets/dxDateBox/1%20Configuration/pickerType.md '/Documentation/ApiReference/UI_Widgets/dxDateBox/Configuration/#pickerType') is *"list"*.
