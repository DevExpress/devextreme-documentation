id: dxDateBox.Options.showAnalogClock
type: Boolean
default: true
---
---
##### shortDescription
Specifies whether to show the analog clock in the value picker. Applies only if [type](/api-reference/10%20UI%20Widgets/dxDateBox/1%20Configuration/type.md '/Documentation/ApiReference/UI_Widgets/dxDateBox/Configuration/#type') is *"datetime"* and [pickerType](/api-reference/10%20UI%20Widgets/dxDateBox/1%20Configuration/pickerType.md '/Documentation/ApiReference/UI_Widgets/dxDateBox/Configuration/#pickerType') is *"calendar"*.
