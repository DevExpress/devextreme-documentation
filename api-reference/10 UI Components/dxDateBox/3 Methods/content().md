##### shortDescription
Gets the popup window's content.

##### return: DxElement
An <a href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement" target="_blank">HTML element</a> or a <a href="http://api.jquery.com/Types/#jQuery" target="_blank">jQuery element</a> when you use jQuery.

---

[note] The **content()** method always returns `null` if [pickerType](/Documentation/ApiReference/UI_Components/dxDateBox/Configuration/#pickerType) is *"native"*.
