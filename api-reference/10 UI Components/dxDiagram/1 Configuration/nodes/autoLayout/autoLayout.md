##### shortDescription
Specifies an auto-layout algorithm that the UI component uses to build a diagram.

---
The **layout** property is in effect when a diagram is bound to a data source using the [nodes](/api-reference/10%20UI%20Widgets/dxDiagram/1%20Configuration/nodes '/Documentation/ApiReference/UI_Components/dxDiagram/Configuration/nodes/') property.
