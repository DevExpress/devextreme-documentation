---
id: dxDiagram.Options.nodes.autoSizeEnabled
type: Boolean
default: true
---
---
##### shortDescription
Specifies whether or not a shape size is automatically changed to fit the text when the UI component is bound to a data source.

---
[note] If the [widthExpr](/api-reference/10%20UI%20Components/dxDiagram/1%20Configuration/nodes/widthExpr.md '/Documentation/ApiReference/UI_Components/dxDiagram/Configuration/nodes/#widthExpr') and [heightExpr](/api-reference/10%20UI%20Components/dxDiagram/1%20Configuration/nodes/heightExpr.md '/Documentation/ApiReference/UI_Components/dxDiagram/Configuration/nodes/#heightExpr') options are specified, the autosize feature is disabled. The UI component displays shapes based on the provided sizes.
