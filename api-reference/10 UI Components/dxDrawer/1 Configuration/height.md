##### merge

##### shortDescription
Specifies the view's height.

---
This property accepts a value of one of the following types:

- **Number**  
The height in pixels.

- **String**  
A CSS-accepted measurement of height. For example, `"55px"`, `"20vh"`, `"inherit"`. If you want to specify a relative height, use `vh` height measurements.

- **Function** (deprecated since v21.2)         
Refer to the [W0017](/Documentation/ApiReference/Common/Utils/Errors_and_Warnings/#W0017) warning description for information on how to migrate to viewport units.
