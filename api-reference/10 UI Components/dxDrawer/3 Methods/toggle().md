id: dxDrawer.toggle()

---
##### shortDescription
Opens or closes the drawer, reversing the current state.

##### return: Promise<void>
A Promise that is resolved after the drawer is opened or closed. It is a <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise" target="_blank">native Promise</a> or a <a href="http://api.jquery.com/Types/#Promise" target="_blank">jQuery.Promise</a> when you use jQuery.
