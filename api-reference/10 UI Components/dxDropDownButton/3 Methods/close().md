---
id: dxDropDownButton.close()
---
---
##### shortDescription
Closes the drop-down menu.

##### return: Promise<void>
A Promise that is resolved when the drop-down menu is closed. It is a <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise" target="_blank">native Promise</a> or a <a href="http://api.jquery.com/Types/#Promise" target="_blank">jQuery.Promise</a> when you use jQuery.
