---
id: dxFileManager.Options.upload.chunkSize
type: Number
default: 200000
---
---
##### shortDescription
Specifies a chunk size in bytes.

---

The FileManager can divide a large file in parts and upload them in separate requests. 

---

##### jQuery

    $(function () {
        $("#file-manager").dxFileManager({
            upload: {
                chunkSize: 500000
            },
            // ...
        });
    });
