##### shortDescription
The error code.

---
The following error codes are supported:

- NoAccess = 0
- FileExists = 1
- FileNotFound = 2
- DirectoryExists = 3
- DirectoryNotFound = 4
- WrongFileExtension = 5
- MaxFileSizeExceeded = 6
- Other = 32767
