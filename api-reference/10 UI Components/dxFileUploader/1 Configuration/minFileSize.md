##### shortDescription
Specifies the minimum file size (in bytes) allowed for uploading. Applies only if [uploadMode](/api-reference/10%20UI%20Components/dxFileUploader/1%20Configuration/uploadMode.md '/Documentation/ApiReference/UI_Components/dxFileUploader/Configuration/#uploadMode') is *"instantly*" or *"useButtons"*.
