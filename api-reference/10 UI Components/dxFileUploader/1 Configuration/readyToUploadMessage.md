##### shortDescription
The message displayed by the UI component when it is ready to upload the specified files.

---
This property makes sense only if the [uploadMode](/api-reference/10%20UI%20Widgets/dxFileUploader/1%20Configuration/uploadMode.md '/Documentation/ApiReference/UI_Components/dxFileUploader/Configuration/#uploadMode') property is set to "useButtons".
