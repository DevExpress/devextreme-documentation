##### shortDescription
Specifies whether current editor values differ from initial values.

---
This property is a read-only flag. You can use it to check if any editor value changed on this Form. 

[note] Specify the [dataField](/Documentation/ApiReference/UI_Components/dxForm/Item_Types/SimpleItem/#dataField) of each Form item for this property to take effect.
