##### shortDescription
The text displayed for optional fields. Applies only if [showOptionalMark](/Documentation/ApiReference/UI_Components/dxForm/Configuration/#showOptionalMark) is **true**.

---
[note] In *"static"* or *"floating"* [labelMode](/Documentation/ApiReference/UI_Components/dxForm/Configuration/#labelMode), form items use the **optionalMark** property value as help text if the [helpText](/Documentation/ApiReference/UI_Components/dxForm/Item_Types/SimpleItem/#helpText) property is not specified.
