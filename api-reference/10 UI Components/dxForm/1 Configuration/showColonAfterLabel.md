##### shortDescription
Specifies whether a colon is displayed at the end of form labels. Applies only to labels outside their editors (see [labelMode](/api-reference/10%20UI%20Components/dxForm/1%20Configuration/labelMode.md '/Documentation/ApiReference/UI_Components/dxForm/Configuration/#labelMode')).
