##### shortDescription
Shows a specific image.

##### return: Promise<void>
A Promise that is resolved after the image is shown.
#include ref-promisedistinction

##### param(itemIndex): Number
The index of the image in the [dataSource](/api-reference/10%20UI%20Components/dxGallery/1%20Configuration/dataSource.md '/Documentation/ApiReference/UI_Components/dxGallery/Configuration/#dataSource').

##### param(animation): Boolean
Specifies whether switching to the image should be animated.
