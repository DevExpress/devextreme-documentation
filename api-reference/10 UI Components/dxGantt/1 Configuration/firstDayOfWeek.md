##### shortDescription
Specifies the first day of a week.

---
The option's value can be from 0 to 6.

- 0 - Sunday
- 1 - Monday
- 2 - Tuesday
- 3 - Wednesday
- 4 - Thursday
- 5 - Friday
- 6 - Saturday

The culture settings specify the option's default value.
