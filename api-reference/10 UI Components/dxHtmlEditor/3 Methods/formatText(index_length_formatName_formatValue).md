##### shortDescription
Applies a single [text format](/concepts/05%20UI%20Components/HtmlEditor/10%20Formats '/Documentation/Guide/UI_Components/HtmlEditor/Formats/') to all characters in the given range.

<!-- %param(index)% -->

##### param(length): Number
<!-- %param(length)% -->

<!-- %param(formatName)% -->

##### param(formatValue): any
<!-- %param(formatValue)% -->
