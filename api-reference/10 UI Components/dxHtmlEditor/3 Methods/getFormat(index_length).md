id: dxHtmlEditor.getFormat(index, length)
---
---
##### shortDescription
Gets formats applied to the content in the specified range.

##### return: Object
The applied formats.         
This object has the following structure:        
`{ "formatName1": "formatValue1", ... }`

##### param(index): Number
A zero-based index indicating the range's start.

##### param(length): Number
The range's length.     
[Embedded items](/concepts/05%20Widgets/HtmlEditor/10%20Formats '/Documentation/Guide/Widgets/HtmlEditor/Formats/') have a length of 1.
