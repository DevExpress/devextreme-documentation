---
id: dxHtmlEditor.insertEmbed(index, type, config)
---
---
##### shortDescription
Inserts an embedded content at the specified position.

##### param(index): Number
A zero-based index at which to insert an embedded content.

##### param(type): String
An embedded format's [name](/concepts/05%20Widgets/HtmlEditor/10%20Formats '/Documentation/Guide/UI_Components/HtmlEditor/Formats/').

##### param(config): any
An embedded format's [value](/concepts/05%20Widgets/HtmlEditor/10%20Formats '/Documentation/Guide/UI_Components/HtmlEditor/Formats/').

---
---
##### jQuery

    <!--JavaScript-->
    // Adds an image at the beginning of the content
    $("#htmlEditorContainer").dxHtmlEditor("instance").insertEmbed(0, "extendedImage", {
        src: "https://js.devexpress.com/Content/images/doc/20_1/PhoneJS/person1.png",
        alt: "Photo",
        width: "100px"
    });
