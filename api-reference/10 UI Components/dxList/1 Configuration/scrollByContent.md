---
id: dxList.Options.scrollByContent
type: Boolean
default: true, false (non-touch_devices)
---
---
##### shortDescription
Specifies whether users can swipe List content to scroll.

---

[note] When **scrollByContent** is enabled, List does not allow text selection within items.
