##### shortDescription
Collapses a group with a specific index.

##### return: Promise<void>
A Promise that is resolved if the group is collapsed and rejected otherwise. It is a <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise" target="_blank">native Promise</a> or a <a href="http://api.jquery.com/Types/#Promise" target="_blank">jQuery.Promise</a> when you use jQuery.

##### param(groupIndex): Number
The group's index.
