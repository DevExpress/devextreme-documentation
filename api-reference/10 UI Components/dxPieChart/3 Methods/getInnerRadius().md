##### shortDescription
Gets the radius of the doughnut hole in pixels. Applies only when the [type](/api-reference/20%20Data%20Visualization%20Widgets/dxPieChart/1%20Configuration/type.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxPieChart/Configuration/#type') is *"doughnut"* or *"donut"*.

##### return: Number
The radius of the doughnut hole.
