---
id: dxPivotGrid.Options.allowFiltering
type: Boolean
default: false
---
---
##### shortDescription
Allows a user to filter fields by selecting or deselecting values in the popup menu.

---
A user can click a filter icon in the [field chooser](/concepts/05%20UI%20Components/PivotGrid/010%20Visual%20Elements/10%20Field%20Chooser/01%20Field%20Chooser.md '/Documentation/Guide/UI_Components/PivotGrid/Visual_Elements/#Field_Chooser') or [field panel](/concepts/05%20UI%20Components/PivotGrid/010%20Visual%20Elements/15%20Field%20Panel.md '/Documentation/Guide/UI_Components/PivotGrid/Visual_Elements/#Field_Panel') when this property is set to **true** to invoke a popup menu.
