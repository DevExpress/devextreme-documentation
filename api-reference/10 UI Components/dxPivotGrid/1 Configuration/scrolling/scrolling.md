##### shortDescription
A configuration object specifying scrolling options.

---
The **PivotGrid** UI component enables an end-user to scroll grid records. To specify required scrolling behavior, use the [mode](/api-reference/10%20UI%20Widgets/dxPivotGrid/1%20Configuration/scrolling/mode.md '/Documentation/ApiReference/UI_Widgets/dxPivotGrid/Configuration/scrolling/#mode') property of the **scrolling** configuration object.
