##### shortDescription
Cancels changes made in the UI component without applying them to the PivotGrid. Takes effect only if [applyChangesMode](/api-reference/10%20UI%20Components/dxPivotGridFieldChooser/1%20Configuration/applyChangesMode.md '/Documentation/ApiReference/UI_Components/dxPivotGridFieldChooser/Configuration/#applyChangesMode') is *"onDemand"*.
