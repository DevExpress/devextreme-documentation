##### shortDescription
An object that specifies configuration options for all series of the *area* type in the chart.

---
Define a common configuration for all _area_ series in the chart using any options of the [AreaSeries](/api-reference/10%20UI%20Components/dxPolarChart/5%20Series%20Types/AreaSeries '/Documentation/ApiReference/UI_Components/dxPolarChart/Series_Types/AreaSeries/') object.
