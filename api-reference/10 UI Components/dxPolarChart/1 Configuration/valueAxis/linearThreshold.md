##### shortDescription
Specifies a value used to calculate the range on a [logarithmic axis](/api-reference/10%20UI%20Components/dxPolarChart/1%20Configuration/valueAxis/type.md '{basewidgetpath}/Configuration/valueAxis/#type') within which the axis should be linear. Applies only if the data source contains negative values or zeroes.
