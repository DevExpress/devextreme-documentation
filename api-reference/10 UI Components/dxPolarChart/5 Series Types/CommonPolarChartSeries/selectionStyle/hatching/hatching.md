##### shortDescription
Specifies the hatching properties to be applied when a series is selected.

##### propertyOf
dxPolarChartSeriesTypes.areapolarseries,dxPolarChartSeriesTypes.barpolarseries,dxPolarChartSeriesTypes.stackedbarpolarseries

---
Within this object, you can specify how to apply hatching using the **direction** property, the width and opacity of lines with the **width** and **opacity** properties, and the space between the lines using the **step** property.
