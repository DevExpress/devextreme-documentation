##### shortDescription
An object defining the [label](/concepts/05%20UI%20Components/PolarChart/10%20Visual%20Elements/070%20Axis%20Labels.md '/Documentation/Guide/UI_Components/PolarChart/Visual_Elements/#Axis_Labels') configuration properties that are common for all axes in the PolarChart UI component.
