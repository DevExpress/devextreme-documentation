##### shortDescription
Configures UI component visibility animations. This object contains two fields: [show](/api-reference/10%20UI%20Widgets/dxPopover/1%20Configuration/animation/show.md '/Documentation/ApiReference/UI_Widgets/dxPopover/Configuration/animation/#show') and [hide](/api-reference/10%20UI%20Widgets/dxPopover/1%20Configuration/animation/hide.md '/Documentation/ApiReference/UI_Widgets/dxPopover/Configuration/animation/#hide').

---
Set this object to **null** or **undefined** to disable animation.
