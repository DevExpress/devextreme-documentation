---
id: dxPopover.Options.shading
type: Boolean
default: false
---
Shading is applied to the first element with a <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/position" target="_blank">position</a> value different from `static`. The sequence is the following: [container](/api-reference/10%20UI%20Widgets/dxPopover/1%20Configuration/container.md '/Documentation/ApiReference/UI_Widgets/dxPopover/Configuration/#container') => parent elements => **window**.
