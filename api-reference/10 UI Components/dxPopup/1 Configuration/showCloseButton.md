type: Boolean

##### shortDescription
Specifies whether or not the UI component displays the Close button.

---
[note]The option makes sense only if the [showTitle](/api-reference/10%20UI%20Widgets/dxPopup/1%20Configuration/showTitle.md '/Documentation/ApiReference/UI_Components/dxPopup/Configuration/#showTitle') option is set to **true**.
