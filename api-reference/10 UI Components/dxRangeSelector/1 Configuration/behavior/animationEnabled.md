##### shortDescription
Indicates whether or not animation is enabled.

---
To make a selected range and [chart series](/api-reference/10%20UI%20Components/dxRangeSelector/1%20Configuration/chart '/Documentation/ApiReference/UI_Components/dxRangeSelector/Configuration/chart/') move smoothly when required, enable animation by setting the **animationEnabled** property to **true**.
