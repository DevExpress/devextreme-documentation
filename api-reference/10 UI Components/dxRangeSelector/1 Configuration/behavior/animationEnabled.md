##### shortDescription
Indicates whether or not animation is enabled.

---
To make a selected range and [chart series](/api-reference/20%20Data%20Visualization%20Widgets/dxRangeSelector/1%20Configuration/chart '/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/chart/') move smoothly when required, enable animation by setting the **animationEnabled** property to **true**.
