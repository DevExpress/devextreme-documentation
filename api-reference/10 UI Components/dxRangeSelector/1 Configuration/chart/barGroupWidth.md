##### shortDescription
Specifies a fixed width for groups of bars with the same argument, measured in pixels. Takes precedence over the [barGroupPadding](/api-reference/10%20UI%20Components/dxRangeSelector/1%20Configuration/chart/barGroupPadding.md '/Documentation/ApiReference/UI_Components/dxRangeSelector/Configuration/chart/#barGroupPadding') option.

##### propertyOf
