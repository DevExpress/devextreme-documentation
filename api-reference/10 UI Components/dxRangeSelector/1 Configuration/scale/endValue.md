##### shortDescription
Specifies the scale's end value.

---
Setting the start and end values is enough for the RangeSelector UI component to create a scale and arrange ticks automatically. If required, you can also set [major](/api-reference/20%20Data%20Visualization%20Widgets/dxRangeSelector/1%20Configuration/scale/tickInterval '/Documentation/ApiReference/UI_Components/dxRangeSelector/Configuration/scale/tickInterval/') and [minor](/api-reference/20%20Data%20Visualization%20Widgets/dxRangeSelector/1%20Configuration/scale/minorTickInterval '/Documentation/ApiReference/UI_Components/dxRangeSelector/Configuration/scale/minorTickInterval/') tick intervals.
