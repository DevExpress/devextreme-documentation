##### shortDescription
Specifies a value used to calculate the range on a [logarithmic scale](/api-reference/10%20UI%20Components/dxRangeSelector/1%20Configuration/scale/type.md '{basewidgetpath}/Configuration/scale/#type') within which the scale should be linear. Applies only if the data source contains negative values or zeroes.
