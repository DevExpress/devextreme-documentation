---
id: dxRangeSelector.Options.scale.minorTickCount
type: Number
default: undefined
---
---
##### shortDescription
Specifies the number of minor ticks between neighboring major ticks.

---
If you set both the **minorTickCount** property and the [minorTickInterval](/api-reference/20%20Data%20Visualization%20Widgets/dxRangeSelector/1%20Configuration/scale/minorTickInterval '/Documentation/ApiReference/UI_Components/dxRangeSelector/Configuration/scale/minorTickInterval/') property, the **minorTickCount** property value will be ignored.
