type: Number

---
##### shortDescription
Specifies the distance in pixels from the edge of viewport at which scrolling should start. Applies only if [autoScroll](/api-reference/10%20UI%20Widgets/dxScheduler/1%20Configuration/appointmentDragging/autoScroll.md '/Documentation/ApiReference/UI_Widgets/dxScheduler/Configuration/appointmentDragging/#autoScroll') is **true**.
