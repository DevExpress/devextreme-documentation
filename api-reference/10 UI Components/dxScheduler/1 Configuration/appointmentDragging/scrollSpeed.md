##### shortDescription
Specifies the scrolling speed when dragging an appointment beyond the viewport. Applies only if [autoScroll](/api-reference/10%20UI%20Widgets/dxScheduler/1%20Configuration/appointmentDragging/autoScroll.md '/Documentation/ApiReference/UI_Widgets/dxScheduler/Configuration/appointmentDragging/#autoScroll') is **true**.
