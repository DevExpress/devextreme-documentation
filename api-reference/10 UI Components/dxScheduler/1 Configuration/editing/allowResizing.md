##### shortDescription
Specifies whether or not an end user can change an appointment duration.

[note] Resizing a [recurring appointment](https://js.devexpress.com/Demos/WidgetsGallery/Demo/Scheduler/RecurringAppointments/) and choosing "Edit Series" in the "Edit Recurring Appointment" popup changes the start date of the recurring appointment series. The component removes all appointments before the modified appointment.
