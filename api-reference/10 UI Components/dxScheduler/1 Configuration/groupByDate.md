---
id: dxScheduler.Options.groupByDate
type: Boolean
default: false
---
---
##### shortDescription
If **true**, appointments are grouped by date first and then by resource; opposite if **false**. Applies only if appointments are [grouped](/concepts/05%20Widgets/Scheduler/040%20Resources/030%20Group%20Appointments%20by%20Resources.md '/Documentation/Guide/Widgets/Scheduler/Resources/Group_Appointments_by_Resources/') and [groupOrientation](/api-reference/10%20UI%20Widgets/dxScheduler/1%20Configuration/views/groupOrientation.md '/Documentation/ApiReference/UI_Widgets/dxScheduler/Configuration/views/#groupOrientation') is *"horizontal"*.
