##### shortDescription
Configures scrolling.

---
Scrolling allows a user to browse data outside the current viewport. Information about the available scrolling modes is in the [mode](/api-reference/_hidden/dxSchedulerScrolling/mode.md '/Documentation/ApiReference/UI_Components/dxScheduler/Configuration/scrolling/#mode') property description.
