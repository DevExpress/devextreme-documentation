##### shortDescription
Configures scrolling.

---
Scrolling allows a user to browse data left outside the current viewport. The UI component contains two scrolling modes detailed in the [mode](/Documentation/ApiReference/UI_Components/dxScheduler/Configuration/scrolling/#mode) property description.
