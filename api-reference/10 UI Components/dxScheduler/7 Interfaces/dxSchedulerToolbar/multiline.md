---
id: dxSchedulerToolbar.multiline
type: Boolean
default: false
---
---
##### shortDescription
Specifies whether the toolbar arranges items into multiple lines when their combined width exceeds the toolbar width.

---
Note that if you set the **multiline** property to `true`, you cannot use the following [item](/api-reference/10%20UI%20Components/dxScheduler/7%20Interfaces/dxSchedulerToolbar/items.md '/Documentation/ApiReference/UI_Components/dxScheduler/Configuration/toolbar/items/') properties:
