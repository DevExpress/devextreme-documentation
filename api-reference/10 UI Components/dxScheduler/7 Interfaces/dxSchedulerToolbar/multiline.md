---
id: dxSchedulerToolbar.multiline
type: Boolean
default: false
---
---
##### shortDescription
Specifies whether or not the toolbar arranges items into multiple lines when their combined width exceeds the toolbar width.

---
Note that if you set the **multiline** property to `true`, you cannot use the following [item](/Documentation/ApiReference/UI_Components/dxScheduler/Configuration/toolbar/items/) properties:

- [location](/Documentation/ApiReference/UI_Components/dxScheduler/Configuration/toolbar/items/#location)

- [locateInMenu](/Documentation/ApiReference/UI_Components/dxScheduler/Configuration/toolbar/items/#locateInMenu)

- [menuItemTemplate](/Documentation/ApiReference/UI_Components/dxScheduler/Configuration/toolbar/items/#menuItemTemplate)

- [menuItemRender](/Documentation/ApiReference/UI_Components/dxScheduler/Configuration/toolbar/items/#menuItemRender)

- [menuItemComponent](/Documentation/ApiReference/UI_Components/dxScheduler/Configuration/toolbar/items/#menuItemComponent)
