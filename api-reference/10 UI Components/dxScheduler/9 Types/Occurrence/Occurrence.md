##### shortDescription
An object that contains information about an appointment occurrence. The [getOccurrences](/api-reference/10%20UI%20Components/dxScheduler/3%20Methods/getOccurrences(startDate_endDate_appointments).md '/Documentation/ApiReference/UI_Components/dxScheduler/Methods/#getOccurrencesstartDate_endDate_appointments') method returns objects of this type.

---
An occurrence is an appointment instance that falls within the specified interval. Recurring appointments can generate multiple occurrences.
