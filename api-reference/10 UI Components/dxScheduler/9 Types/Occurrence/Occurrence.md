##### shortDescription
An object that contains information about an appointment occurrence. The [getOccurrences](/Documentation/ApiReference/UI_Components/dxScheduler/Methods/#getOccurrencesstartDateendDateappointments) method returns objects of this type.

---
An occurrence is an appointment instance that falls within the specified time interval. If an appointment has a recurrence rule, it can generate multiple occurrences.
