##### shortDescription
An object that contains information about an appointment occurrence. The [getOccurrences](/api-reference/10%20UI%20Components/dxScheduler/3%20Methods/getOccurrences%28startDate_endDate_appointments%29.md '/Documentation/ApiReference/UI_Components/dxScheduler/Methods/#getOccurrences') method returns objects of this type.

---
An occurrence is an appointment instance that falls within the specified time interval. If an appointment has a recurrence rule, it can generate multiple occurrences.
