##### shortDescription
Locks the UI component until the **release(preventScrollBottom)** method is called and executes the function passed to the [onPullDown](/api-reference/10%20UI%20Components/dxScrollView/1%20Configuration/onPullDown.md '/Documentation/ApiReference/UI_Components/dxScrollView/Configuration/#onPullDown') property and the handler assigned to the [pullDown](/api-reference/10%20UI%20Components/dxScrollView/4%20Events/pullDown.md '/Documentation/ApiReference/UI_Components/dxScrollView/Events/#pullDown') event.
