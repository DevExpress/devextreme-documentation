The following values are available for this property:

- *'contains'*    
Searches items that **contain** the user's typed text. The text remains unchanged during search. Auto-fill is not activated.

- *'startswith'*    
Searches items that **start with** the user's typed text. During search, the component auto-fills the input with the first matched item.
