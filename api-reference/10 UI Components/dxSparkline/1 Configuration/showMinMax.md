---
id: dxSparkline.Options.showMinMax
type: Boolean
default: false
---
---
##### shortDescription
Specifies whether or not to indicate both the minimum and maximum values on a sparkline.

---
When this property is set to **true**, the minimum and maximum values are highlighted and painted in the colors specified by the [minColor](/api-reference/10%20UI%20Components/dxSparkline/1%20Configuration/minColor.md '/Documentation/ApiReference/UI_Components/dxSparkline/Configuration/#minColor') and [maxColor](/api-reference/10%20UI%20Components/dxSparkline/1%20Configuration/maxColor.md '/Documentation/ApiReference/UI_Components/dxSparkline/Configuration/#maxColor') properties correspondingly.
