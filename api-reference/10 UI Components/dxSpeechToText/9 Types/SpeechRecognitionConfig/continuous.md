##### shortDescription
Configures the [SpeechRecognition.continuous](https://developer.mozilla.org/en-US/docs/Web/API/SpeechRecognition/continuous) property.

---
The default SpeechRecognition behavior is to raise the [end](/api-reference/10%20UI%20Components/dxSpeechToText/4%20Events/end.md '/Documentation/ApiReference/UI_Components/dxSpeechToText/Events/#end') event when users pause their speech. To continue transcribing speech when users pause, set this property to `true`.
