##### shortDescription
Specifies the size of a smaller side of a separator bar.

---
This property accepts a value of one of the following types:

- **Number**  
The width/height in pixels.

- **String**  
A CSS-accepted measurement of width/height. For example, `"55px"`, `"20vw"`, `"80%"`, `"auto"`, `"inherit"`.
