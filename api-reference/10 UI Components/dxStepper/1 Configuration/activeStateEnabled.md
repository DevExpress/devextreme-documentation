##### shortDescription
Specifies whether Stepper changes its visual state as a result of user interaction.

---
The UI component switches to the active state when users press down the primary mouse button. If this property is set to `true`, the CSS rules for the active state apply. You can change these rules to customize the component.

Use this property on platforms that require the active state change for UI components.
