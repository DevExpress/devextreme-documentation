##### shortDescription
Specifies a navigation mode (linear/non-linear).

---
If this property is `false`, navigation is non-linear. In other words, users can skip steps. 

[note] You cannot skip a [disabled](/api-reference/_hidden/CollectionWidgetItem/disabled.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#disabled') step in linear mode.
