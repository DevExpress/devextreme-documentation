---
id: dxTextArea.Options.autoResizeEnabled
type: Boolean
default: false
---
---
##### shortDescription
A Boolean value specifying whether or not the auto resizing mode is enabled.

---
Use the [minHeight](/api-reference/10%20UI%20Components/dxTextArea/1%20Configuration/minHeight.md '/Documentation/ApiReference/UI_Components/dxTextArea/Configuration/#minHeight') and [maxHeight](/api-reference/10%20UI%20Components/dxTextArea/1%20Configuration/maxHeight.md '/Documentation/ApiReference/UI_Components/dxTextArea/Configuration/#maxHeight') properties to set the height that the adapted TextArea can occupy. **minHeight** specifies the initial height. For more information, refer to the following article: [Resize TextArea](/concepts/05%20UI%20Components/TextArea/00%20Getting%20Started%20with%20TextArea/15%20Resize%20TextArea.md '/Documentation/Guide/UI_Components/TextArea/Getting_Started_with_TextArea/#Resize_TextArea').

Note that the [height](/api-reference/10%20UI%20Components/dxTextArea/1%20Configuration/height.md '/Documentation/ApiReference/UI_Components/dxTextArea/Configuration/#height') property is not in effect if **autoResizeEnabled** is set to **true**.
