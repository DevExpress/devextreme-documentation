---
id: dxTextArea.Options.autoResizeEnabled
type: Boolean
default: false
---
---
##### shortDescription
A Boolean value specifying whether or not the auto resizing mode is enabled.

---
Setting this property makes sense only when the [height](/api-reference/10%20UI%20Components/DOMComponent/1%20Configuration/height.md '/Documentation/ApiReference/UI_Components/dxTextArea/Configuration/#height') property is not specified.
