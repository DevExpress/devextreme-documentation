---
id: dxTextArea.Options.autoResizeEnabled
type: Boolean
default: false
---
---
##### shortDescription
A Boolean value specifying whether or not the auto resizing mode is enabled.

---
Use the [minHeight](/api-reference/10%20UI%20Components/dxTextArea/1%20Configuration/minHeight.md '/Documentation/ApiReference/UI_Components/dxTextArea/Configuration/#minHeight') and [maxHeight](/api-reference/10%20UI%20Components/dxTextArea/1%20Configuration/maxHeight.md '/Documentation/ApiReference/UI_Components/dxTextArea/Configuration/#maxHeight') properties to set the height that the adapted TextArea can occupy. **minHeight** specifies the initial height. For more information, refer to the following article: [Adapt the Size of the Text Area](/concepts/05%20UI%20Components/TextArea/05%20Adapt%20the%20Size%20of%20the%20Text%20Area.md '/Documentation/Guide/UI_Components/TextArea/Adapt_the_Size_of_the_Text_Area/').

Note that the [height](/api-reference/10%20UI%20Components/DOMComponent/1%20Configuration/height.md '/Documentation/ApiReference/UI_Components/dxTextArea/Configuration/#height') property is not in effect if **autoResizeEnabled** is set to **true**.
