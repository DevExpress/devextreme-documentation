##### shortDescription
Specifies the minimum height of the UI component.

---
This property accepts a value of one of the following types:

- **Number**  
The height in pixels.

- **String**  
A CSS-accepted measurement of height. For example, `"55px"`, `"20vh"`, `"80%"`, `"inherit"`.
