---
id: dxToolbar.Options.multiline
type: Boolean
default: false
---
---
##### shortDescription
Specifies whether or not the Toolbar arranges items into multiple lines when their combined width exceeds the Toolbar width.

---
Note that if you set the **multiline** property to `true`, you cannot use the following [item](/Documentation/ApiReference/UI_Components/dxToolbar/Configuration/items/) properties:

- [location](/Documentation/ApiReference/UI_Components/dxToolbar/Configuration/items/#location)

- [menuItemTemplate](/Documentation/ApiReference/UI_Components/dxToolbar/Configuration/items/#menuItemTemplate)

- [menuItemRender](/Documentation/ApiReference/UI_Components/dxToolbar/Configuration/items/#menuItemRender)

- [menuItemComponent](/Documentation/ApiReference/UI_Components/dxToolbar/Configuration/items/#menuItemComponent)
