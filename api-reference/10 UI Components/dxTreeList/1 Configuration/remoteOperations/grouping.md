##### shortDescription
Specifies whether grouping should be performed on the server.

---
Grouping is required only when a user can filter data using a [header filter](/api-reference/10%20UI%20Widgets/dxTreeList/1%20Configuration/headerFilter '/Documentation/ApiReference/UI_Widgets/dxTreeList/Configuration/headerFilter/').
