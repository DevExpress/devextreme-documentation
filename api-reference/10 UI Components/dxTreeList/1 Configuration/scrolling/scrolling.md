##### shortDescription
Configures scrolling.

---
Scrolling allows a user to browse data left outside the current viewport. The UI component provides several scrolling modes detailed in the [mode](/api-reference/10%20UI%20Components/dxTreeList/9%20Types/Scrolling/mode.md '{basewidgetpath}/Configuration/scrolling/#mode') property description.
