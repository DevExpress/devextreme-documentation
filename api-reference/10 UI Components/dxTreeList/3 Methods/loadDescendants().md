##### shortDescription
Loads all root node descendants (all data items). Takes effect only if data has the [plain structure](/api-reference/10%20UI%20Components/dxTreeList/1%20Configuration/dataStructure.md '/Documentation/ApiReference/UI_Components/dxTreeList/Configuration/#dataStructure') and  **remoteOperations**.[filtering](/api-reference/10%20UI%20Components/dxTreeList/1%20Configuration/remoteOperations/filtering.md '/Documentation/ApiReference/UI_Components/dxTreeList/Configuration/remoteOperations/#filtering') is **true**.

##### return: Promise<void>
A Promise that is resolved after data is loaded.
