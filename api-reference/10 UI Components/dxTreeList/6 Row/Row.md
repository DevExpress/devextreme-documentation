##### shortDescription
A grid row.

---
A row is a visual representation of a [node](/api-reference/10%20UI%20Components/dxTreeList/4%20Node '/Documentation/ApiReference/UI_Components/dxTreeList/Node/'). You can access grid rows using the [getVisibleRows()](/api-reference/10%20UI%20Components/dxTreeList/3%20Methods/getVisibleRows().md '/Documentation/ApiReference/UI_Components/dxTreeList/Methods/#getVisibleRows') method and within some event handlers, for example, [onCellClick](/api-reference/10%20UI%20Components/dxTreeList/1%20Configuration/onCellClick.md '/Documentation/ApiReference/UI_Components/dxTreeList/Configuration/#onCellClick'), [onCellPrepared](/api-reference/10%20UI%20Components/dxTreeList/1%20Configuration/onCellPrepared.md '/Documentation/ApiReference/UI_Components/dxTreeList/Configuration/#onCellPrepared'), [onEditorPreparing](/api-reference/10%20UI%20Components/dxTreeList/1%20Configuration/onEditorPreparing.md '/Documentation/ApiReference/UI_Components/dxTreeList/Configuration/#onEditorPreparing').
