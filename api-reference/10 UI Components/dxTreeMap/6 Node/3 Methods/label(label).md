id: dxTreeMapNode.label(label)
---
---
##### shortDescription
Sets the label to the node.

##### param(label): String
A new label.

---
Usually, labels are provided by a [data source field](/api-reference/10%20UI%20Components/dxTreeMap/1%20Configuration/labelField.md '/Documentation/ApiReference/UI_Components/dxTreeMap/Configuration/#labelField'). To change the label of a specific node, call the **label** method passing the new text for the label as its parameter.
