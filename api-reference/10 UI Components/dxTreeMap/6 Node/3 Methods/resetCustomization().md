##### shortDescription
Reverts the appearance of the node to the initial state.

---
Use this method if you need to revert the initial appearance after customizing it using the [customize(options)](/api-reference/10%20UI%20Components/dxTreeMap/6%20Node/3%20Methods/customize(options).md '/Documentation/ApiReference/UI_Components/dxTreeMap/Node/Methods/#customizeoptions') method.
