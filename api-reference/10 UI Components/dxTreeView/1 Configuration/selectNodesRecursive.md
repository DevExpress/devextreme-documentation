id: dxTreeView.Options.selectNodesRecursive
type: Boolean
default: true

---
##### shortDescription
Specifies whether all child nodes should be selected when their parent node is selected. Applies only if the [selectionMode](/api-reference/10%20UI%20Widgets/dxTreeView/1%20Configuration/selectionMode.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#selectionMode') is *"multiple"*.
