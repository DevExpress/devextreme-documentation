##### shortDescription
Resets the value and validation result of the editors that are included to the current validation group.

---
This method calls the [reset](/api-reference/10%20UI%20Components/dxValidator/3%20Methods/reset().md '/Documentation/ApiReference/UI_Components/dxValidator/Methods/#reset') method of each [Validator](/api-reference/10%20UI%20Components/dxValidator '/Documentation/ApiReference/UI_Components/dxValidator/') object included to the current validation group.
