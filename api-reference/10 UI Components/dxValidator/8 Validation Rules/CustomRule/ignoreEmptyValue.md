##### shortDescription
If **true**, the [validationCallback](/api-reference/10%20UI%20Widgets/dxValidator/8%20Validation%20Rules/CustomRule/validationCallback.md '/Documentation/ApiReference/UI_Widgets/dxValidator/Validation_Rules/CustomRule/#validationCallback') is not executed for **null**, **undefined**, **false**, and empty strings.
