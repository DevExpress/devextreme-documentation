##### shortDescription
A validation rule that demands that the validated field match a specified pattern.

---
To specify the regular expression that the validated field must match, set the rule's [pattern](/api-reference/10%20UI%20Components/dxValidator/8%20Validation%20Rules/PatternRule/pattern.md '/Documentation/ApiReference/UI_Components/dxValidator/Validation_Rules/PatternRule/#pattern') configuration option.
