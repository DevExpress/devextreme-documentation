##### shortDescription
A validation rule that demands the target value length be within the specified value range (including the range's end points).

---
To specify the range that the validated value length must match, set the rule's [min](/api-reference/10%20UI%20Components/dxValidator/8%20Validation%20Rules/StringLengthRule/min.md '/Documentation/ApiReference/UI_Components/dxValidator/Validation_Rules/StringLengthRule/#min') and [max](/api-reference/10%20UI%20Components/dxValidator/8%20Validation%20Rules/StringLengthRule/max.md '/Documentation/ApiReference/UI_Components/dxValidator/Validation_Rules/StringLengthRule/#max') configuration properties.

[note]This rule validates string values or the values that can be cast to a string.
