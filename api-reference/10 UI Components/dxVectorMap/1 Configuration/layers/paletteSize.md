##### shortDescription
Specifies the number of colors in a palette.

---
Assign a number that is greater than 0 to this option in order to use [palettes](/api-reference/10%20UI%20Components/dxVectorMap/1%20Configuration/layers/palette.md '/Documentation/ApiReference/UI_Components/dxVectorMap/Configuration/layers/#palette').
