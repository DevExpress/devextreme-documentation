##### shortDescription
Specifies the number of colors in a palette.

---
Assign a number that is greater than 0 to this option in order to use [palettes](/api-reference/20%20Data%20Visualization%20Widgets/dxVectorMap/1%20Configuration/layers/palette.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#palette').
