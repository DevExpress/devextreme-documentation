##### merge

##### shortDescription
Arranges legend items vertically (in a column) or horizontally (in a row). The default value is *"horizontal"* if the **legends[]**.[horizontalAlignment](/api-reference/20%20Data%20Visualization%20Widgets/dxVectorMap/1%20Configuration/legends/horizontalAlignment.md '{basewidgetpath}/Configuration/legends/#horizontalAlignment') is *"center"*. Otherwise, it is *"vertical"*.
