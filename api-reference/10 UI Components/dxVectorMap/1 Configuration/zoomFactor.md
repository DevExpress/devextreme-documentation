##### shortDescription
Specifies a number that is used to zoom a map initially.

---
Use this property to specify a zoom factor for a map while configuring it. This property accepts a value that is greater than 1. Note that the zooming is performed with relation to the [center](/api-reference/20%20Data%20Visualization%20Widgets/dxVectorMap/1%20Configuration/center.md '/Documentation/ApiReference/UI_Components/dxVectorMap/Configuration/#center') of the map.
