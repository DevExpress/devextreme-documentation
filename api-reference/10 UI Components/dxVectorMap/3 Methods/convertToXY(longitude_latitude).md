##### shortDescription
Converts coordinates from the [dataSource](/api-reference/20%20Data%20Visualization%20Widgets/dxVectorMap/1%20Configuration/layers/dataSource.md '/Documentation/ApiReference/UI_Components/dxVectorMap/Configuration/layers/#dataSource') coordinate system to pixels.

##### return: Array<Number>
The point's coordinates in the client area coordinate system.

##### param(longitude): Number
A point's longitude.

##### param(latitude): Number
A point's latitude.
