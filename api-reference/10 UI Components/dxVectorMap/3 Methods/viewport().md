##### shortDescription
Gets the current map viewport coordinates.

##### return: Array<Number>
The viewport coordinates.

---
This method returns the viewport coordinates as an array of four values in the [*minLongitude*, *maxLatitude*, *maxLongitude*, *minLatitude*] form.
