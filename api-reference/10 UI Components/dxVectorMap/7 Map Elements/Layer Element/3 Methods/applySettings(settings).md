---
id: MapLayerElement.applySettings(settings)
---
---
##### shortDescription
Applies the layer element settings and updates element appearance.

##### param(settings): Object
A [configuration object](/api-reference/20%20Data%20Visualization%20Widgets/dxVectorMap/1%20Configuration/layers '/Documentation/ApiReference/UI_Components/dxVectorMap/Configuration/layers/') for the element.

---
The **settings** object should contain one or more properties to be changed for the layer element. The properties available to change using this method are listed in the [layers](/api-reference/20%20Data%20Visualization%20Widgets/dxVectorMap/1%20Configuration/layers '/Documentation/ApiReference/UI_Components/dxVectorMap/Configuration/layers/') property description.

Note that you cannot change the [dataSource](/api-reference/20%20Data%20Visualization%20Widgets/dxVectorMap/1%20Configuration/layers/dataSource.md '/Documentation/ApiReference/UI_Components/dxVectorMap/Configuration/layers/#dataSource') property for a separate layer element.
