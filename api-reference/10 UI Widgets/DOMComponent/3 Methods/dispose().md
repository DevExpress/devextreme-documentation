---
id: DOMComponent.dispose()
---
---
##### shortDescription
Disposes of all the resources allocated to the **{WidgetName}** instance. Use this method only if the widget was created with jQuery or pure JavaScript.

---
After calling this method, remove the DOM element associated with the widget:

    <!--JavaScript-->
    $("#my{WidgetName}").dx{WidgetName}("dispose");
    $("#my{WidgetName}").remove();
