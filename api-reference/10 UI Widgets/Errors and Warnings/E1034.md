##### shortDescription
The current browser does not implement an API required for saving files.

---
This error occurs when your browser version does not implement API for saving files. We recommend that you update your browser to a newer version. Supported browsers are listed [here](/concepts/Common/Introduction%20to%20DevExtreme/05%20Supported%20Browsers.md '/Documentation/Guide/Common/Introduction_to_DevExtreme/#Supported_Browsers').
