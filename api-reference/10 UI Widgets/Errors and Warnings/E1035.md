##### shortDescription
Occurs if the editor cannot be created for a [Form](/api-reference/10%20UI%20Widgets/dxForm '/Documentation/ApiReference/UI_Widgets/dxForm/') item because of an internal error.

Most common cases:

- Incorrect value of the [editorType](/ApiReference/UI_Widgets/dxForm/Item_Types/SimpleItem/#editorType)
- Incorrect initialization of the [editorOptions](/ApiReference/UI_Widgets/dxForm/Item_Types/SimpleItem/#editorOptions)
- Incorrect using of the [empty items](/Guide/Widgets/Form/Organize_Simple_Items/Add_an_Empty_Space/)
- Incorrect using of the [template](/Documentation/ApiReference/UI_Widgets/dxForm/Item_Types/SimpleItem/#template/)
- If you use DevExtreme modules, import the editor's module when specifying the [editorType](/Documentation/ApiReference/UI_Widgets/dxForm/Item_Types/SimpleItem/#editorType)
