##### shortDescription
Occurs if the server returns grouped data that does not match the required format.

---
The [DataGrid](/api-reference/10%20UI%20Widgets/dxDataGrid '/Documentation/ApiReference/UI_Widgets/dxDataGrid/') UI component can display grouped data if data items have the following structure.

    <!--JavaScript-->
    {
        key: "group1", //The group key
        items: [
            . . . //Group items
        ]
    }
