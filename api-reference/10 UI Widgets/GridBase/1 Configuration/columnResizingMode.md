---
##### shortDescription
Specifies how the UI component resizes columns. Applies only if [allowColumnResizing](/api-reference/10%20UI%20Widgets/GridBase/1%20Configuration/allowColumnResizing.md '{basewidgetpath}/Configuration/#allowColumnResizing') is **true**.

---
The **columnResizingMode** option accepts one of the following values:

- **nextColumn**  
    When a user resizes a column, the width of the next column changes.  
- **widget**  
    When a user resizes a column, the width of the UI component changes.  
    This mode is ignored if you specify the [width](/api-reference/_hidden/GridBaseColumn/width.md '{basewidgetpath}/Configuration/columns/#width') of any column in percent.
