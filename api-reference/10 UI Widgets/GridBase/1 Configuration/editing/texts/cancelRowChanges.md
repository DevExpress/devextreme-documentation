##### shortDescription
Specifies text for a button that cancels changes in a row. Applies only if **editing**.[allowUpdating](/api-reference/10%20UI%20Widgets/dxDataGrid/1%20Configuration/editing/allowUpdating.md '{basewidgetpath}/Configuration/editing/#allowUpdating') is **true** and **editing**.[mode](/api-reference/10%20UI%20Widgets/GridBase/1%20Configuration/editing/mode.md '{basewidgetpath}/Configuration/editing/#mode') is *"row"*.
