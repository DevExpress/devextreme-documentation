##### shortDescription
Exports grid data to Excel.

##### param(selectionOnly): Boolean
Specifies whether to export all the data presented in the grid or the selected rows only.

---
For details on exporting, refer to the [Client-Side Exporting](/concepts/05%20Widgets/DataGrid/70%20Client-Side%20Exporting '/Documentation/Guide/Widgets/DataGrid/Client-Side_Exporting/') article.
