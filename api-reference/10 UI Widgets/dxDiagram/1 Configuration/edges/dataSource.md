##### shortDescription
Binds the **edges** collection to the specified data. Specify this option if you use [node and edge](/Documentation/Guide/Widgets/Diagram/Data_Binding/#Node_and_Edge_Arrays) data sources.

---
The Diagram widget creates a connector between two shapes for every edge in the collection.
