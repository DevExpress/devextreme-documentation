type: String | function(data)
default: 'id'

---
##### shortDescription
Specifies the name of a data source field or an expression that provides edge keys.

##### param(data): Object
The current edge's data object.

---
This option is required if you bind edges to a data source ([edges.dataSource](/Documentation/ApiReference/UI_Widgets/dxDiagram/Configuration/edges/#dataSource)).
