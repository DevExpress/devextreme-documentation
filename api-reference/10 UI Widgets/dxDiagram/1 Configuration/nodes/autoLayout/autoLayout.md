##### shortDescription
Specifies an auto-layout algorithm that the widget uses to build a diagram.

---
The **layout** option is in effect when a diagram is bound to a data source using the [nodes](/api-reference/10%20UI%20Widgets/dxDiagram/1%20Configuration/nodes '/Documentation/ApiReference/UI_Widgets/dxDiagram/Configuration/nodes/') option.
