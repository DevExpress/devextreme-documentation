##### shortDescription
Specifies the diagram layout type.

---
When the **type** option is set to **layered** or **tree** the shape coordinates specified by the [leftExpr](/Documentation/ApiReference/UI_Widgets/dxDiagram/Configuration/nodes/#leftExpr) and [topExpr](/Documentation/ApiReference/UI_Widgets/dxDiagram/Configuration/nodes/#topExpr) options are ignored.
