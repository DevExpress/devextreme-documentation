##### shortDescription
Opens or closes the drawer, reversing the current state.

##### return: Promise<void>
A Promise that is resolved after the drawer is opened or closed. It is a [native Promise](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise) or a [jQuery.Promise](https://api.jquery.com/Types/#Promise) when you use jQuery.
