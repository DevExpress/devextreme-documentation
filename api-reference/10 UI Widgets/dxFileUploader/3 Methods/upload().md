##### shortDescription
Uploads all the selected files.

---

[note]
File upload is not supported in **useForms** upload mode ([uploadMode](/Documentation/ApiReference/UI_Widgets/dxFileUploader/Configuration/#uploadMode)). In **instantly** upload mode, use [value](/Documentation/ApiReference/UI_Widgets/dxFileUploader/Configuration/#value) to select a file(s) you want to upload.
[/note]
