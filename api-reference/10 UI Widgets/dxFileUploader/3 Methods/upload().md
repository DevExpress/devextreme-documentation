##### shortDescription
Uploads all selected files.

---

[note]
Upload is not supported in **useForms** upload mode ([uploadMode](/Documentation/ApiReference/UI_Widgets/dxFileUploader/Configuration/#uploadMode)). In **instantly** upload mode, use [value](/Documentation/ApiReference/UI_Widgets/dxFileUploader/Configuration/#value) to select a file(s) for uploading.
[/note]
