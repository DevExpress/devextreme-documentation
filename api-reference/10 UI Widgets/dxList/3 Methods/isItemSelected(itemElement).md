---
id: dxList.isItemSelected(itemElement)
---
---
##### shortDescription
Checks whether an item found using its DOM node is selected.

##### return: Boolean
**true** if the item is selected; otherwise **false**.

##### param(itemElement): Element
The item's DOM node.
