---
id: dxPopover.Options.shading
type: Boolean
default: false
---
Shading applies to the first element with non-static <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/position" target="_blank">position</a> in terms of CSS. The sequence is the following: [container](/Documentation/ApiReference/UI_Widgets/dxPopover/Configuration/#container) => Descendant elements => **window**.

The [target](/Documentation/ApiReference/UI_Widgets/dxPopover/Configuration/#target) option does not affect shading.
