Use the [position](/api-reference/10%20UI%20Widgets/dxPopup/1%20Configuration/position.md '/Documentation/ApiReference/UI_Widgets/dxPopup/Configuration/#position') option to specify the position in which the UI component is shown and from which it is hidden.

Set the **animation** object to **null** or **undefined** to disable animation.
