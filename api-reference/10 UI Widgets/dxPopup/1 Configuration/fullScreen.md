---
id: dxPopup.Options.fullScreen
type: Boolean
default: false
---
---
##### shortDescription
Specifies whether to display the **Popup** in full-screen mode.

---

[note] When **fullScreen** is **true**, the UI component always occupies the entire screen and disregards the [container](/api-reference/10%20UI%20Widgets/dxPopup/1%20Configuration/container.md '/Documentation/ApiReference/UI_Widgets/dxPopup/Configuration/#container') value.
