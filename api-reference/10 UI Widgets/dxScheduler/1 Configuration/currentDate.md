id: dxScheduler.Options.currentDate
type: Date | Number | String
default: new Date()

---
##### shortDescription
Specifies the current date.

---

Users can change this value using the Date Navigator:

![Scheduler Date Navigator](/images/UiWidgets/Scheduler_Date_Navigator.png)

To subscribe to changes of the current date, use the [onOptionChanged](/Documentation/ApiReference/UI_Widgets/dxScheduler/Configuration/#onOptionChanged) option.
