##### shortDescription
Specifies the <a href="https://en.wikipedia.org/wiki/DOM_events" target="_blank">DOM events</a> after which the UI component's [value](/api-reference/10%20UI%20Widgets/dxDropDownList/1%20Configuration/value.md '{basewidgetpath}/Configuration/#value') should be updated. Applies only if [acceptCustomValue](/api-reference/10%20UI%20Widgets/dxSelectBox/1%20Configuration/acceptCustomValue.md '{basewidgetpath}/Configuration/#acceptCustomValue') is set to **true**.
