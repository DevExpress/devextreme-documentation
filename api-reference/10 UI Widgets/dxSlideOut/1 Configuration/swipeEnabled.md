---
id: dxSlideOut.Options.swipeEnabled
type: Boolean
default: true
---
---
##### shortDescription
Indicates whether the menu can be shown/hidden by swiping the widget's main panel.

---
When this option is set to **false**, ensure that you have elements in the widget that call the widget's [showMenu()](/api-reference/10%20UI%20Widgets/dxSlideOut/3%20Methods/showMenu().md '/Documentation/ApiReference/UI_Widgets/dxSlideOut/Methods/#showMenu') and [hideMenu()](/api-reference/10%20UI%20Widgets/dxSlideOut/3%20Methods/hideMenu().md '/Documentation/ApiReference/UI_Widgets/dxSlideOut/Methods/#hideMenu') methods.
