##### shortDescription
Specifies a multiplier for the [baseItemWidth](/api-reference/10%20UI%20Widgets/dxTileView/1%20Configuration/baseItemWidth.md '/Documentation/ApiReference/UI_Widgets/dxTileView/Configuration/#baseItemWidth') option value (for the purpose of obtaining the actual item width).

---
Whether you use a default or a custom template for widget items, their width will be determined according to the value specified for the **widthRatio** field in the item data source object.
