##### shortDescription
Configures scrolling.

---
Scrolling allows a user to browse data left outside the current viewport. The UI component provides several scrolling modes detailed in the [mode](/api-reference/10%20UI%20Widgets/dxTreeList/1%20Configuration/scrolling/mode.md '{basewidgetpath}/Configuration/scrolling/#mode') option description.
