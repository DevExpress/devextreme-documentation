id: dxTreeView.collapseItem(itemElement)
---
---
##### shortDescription
Collapses an item found using its DOM node.

##### param(itemElement): Node
The item's DOM node.

##### return: Boolean
**false** if no node was found or node is disabled or collapsing operation is in progress.
