id: dxTreeView.collapseItem(key)
---
---
##### shortDescription
Collapses an item with a specific key.

##### param(key): any
The item's key.

##### return: Promise<void>
A Promise that is resolved after the TreeView collapses the node and rejected if the node does't exists or is disabled. It is a <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise" target="_blank">native Promise</a> or a <a href="http://api.jquery.com/Types/#Promise" target="_blank">jQuery.Promise</a> when you use jQuery.
