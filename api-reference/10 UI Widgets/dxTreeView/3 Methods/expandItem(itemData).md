##### shortDescription
Expands an item found using its data object.

##### return: Promise<void>
A Promise that is resolved after the item is expanded and rejected if the item is nonexistent or [disabled](/api-reference/10%20UI%20Widgets/dxTreeView/4%20Node/disabled.md '/Documentation/ApiReference/UI_Widgets/dxTreeView/Node/#disabled').             
It is a <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise" target="_blank">native Promise</a> or a <a href="http://api.jquery.com/Types/#Promise" target="_blank">jQuery.Promise</a> when you use jQuery.

##### param(itemData): Object
The item's data object.
