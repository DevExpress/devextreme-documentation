id: dxTreeView.selectItem(key)
---
---
##### shortDescription
Selects an item with a specific key.

##### param(key): any
The item's key.

##### return: Boolean
If the element for the specified key doesn't exist or has not yet loaded, then the method returns false. In other cases, it returns true.
