---
##### shortDescription
Updates the tree view scrollbars according to the current size of the widget content.

##### return: Promise<void>
A Promise that is resolved if the widget is updated and rejected otherwise. It is a <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise" target="_blank">native Promise</a> or a <a href="http://api.jquery.com/Types/#Promise" target="_blank">jQuery.Promise</a> when you use jQuery.
