##### shortDescription
Specifies a range container's background color.

---
<p>The background color is used to indicate the area reserved for the range container in the gauge. This color is replaced with the colors of the ranges contained in the range container.</p>

<p>This option supports the following colors.</p>

<ul>
<li>Hexadecimal colors</li>
<li>RGB colors</li>
<li>RGBA colors</li>
<li>Predefined/Cross-browser color names</li>
<li><a href="http://www.w3.org/TR/SVG/types.html#ColorKeywords">Predefined SVG colors</a></li>
</ul>
