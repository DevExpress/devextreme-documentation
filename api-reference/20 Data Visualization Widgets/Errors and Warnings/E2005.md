##### shortDescription
Occurs in the **Funnel** widget when the specified value field is absent in the data source or all its values are negative.

---
Check that you set the [valueField](/api-reference/20%20Data%20Visualization%20Widgets/dxFunnel/1%20Configuration/valueField.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxFunnel/Configuration/#valueField') option to a field in the data source containing at least one positive value.
