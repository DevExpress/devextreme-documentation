##### shortDescription
Occurs when the visible range specified by the [min](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/argumentAxis/#min) and [max](/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/argumentAxis/#max) options of a chart axis is not valid.

---
The **min** and **max** options accept a numeric or date-time value depending on the [data type](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/argumentAxis/argumentType.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/argumentAxis/#argumentType'). If anything else is passed, the E2106 error occurs.
