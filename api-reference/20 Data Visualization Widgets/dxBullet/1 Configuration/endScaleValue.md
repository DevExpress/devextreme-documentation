##### shortDescription
Specifies an end value for the invisible scale.

---
The primary, target and zero values of the **Bullet** widget are held within a range. This range is represented by an invisible scale. To specify the boundary values of this scale, set the [startScaleValue](/api-reference/20%20Data%20Visualization%20Widgets/dxBullet/1%20Configuration/startScaleValue.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxBullet/Configuration/#startScaleValue') and **endScaleValue** options.

Specify the same start and end scale values for several bullets when you display them in a column so that they can be compared visually.
