##### shortDescription
Specifies a value used to calculate the range on a [logarithmic axis](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/argumentAxis/type.md '{basewidgetpath}/Configuration/argumentAxis/#type') within which the axis should be linear. Applies only if the data source contains negative values or zeroes.
