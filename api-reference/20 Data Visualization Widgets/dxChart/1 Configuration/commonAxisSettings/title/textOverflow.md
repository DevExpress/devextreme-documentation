##### shortDescription
Specifies what to do with the axis title when it overflows the allocated space after applying [wordWrap](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/commonAxisSettings/title/wordWrap.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonAxisSettings/title/#wordWrap'): hide, truncate them and display an ellipsis, or do nothing.
