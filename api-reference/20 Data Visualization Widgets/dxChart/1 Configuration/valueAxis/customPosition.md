---
id: dxChart.Options.valueAxis.customPosition
type: Number | Date | String
default: undefined
---
---
##### shortDescription
Specifies the position of the value axis on the argument axis.

---
The value of this option should be specified in the same format as the arguments on the argument axis. The value axis will stick to the pane border if the **customPosition**'s value is outside the argument axis range.
