##### shortDescription
Specifies the shift in pixels of the value axis.

---
Negative numbers shift the axis position to the left, positive numbers - to the right. If the [rotated](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/rotated.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/#rotated') option is enabled, negative numbers will shift the axis position to the top, and positive numbers - to the bottom.
