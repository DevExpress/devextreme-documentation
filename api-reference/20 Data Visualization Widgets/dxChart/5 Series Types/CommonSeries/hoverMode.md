##### shortDescription
Specifies series elements to be highlighted when a user points to a series.

---
Depending on the series type, this option accepts different values. For information on them, visit the [Series Types](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/5%20Series%20Types '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Series_Types/') section, choose the employed series type and refer to its **hoverMode** option description.
