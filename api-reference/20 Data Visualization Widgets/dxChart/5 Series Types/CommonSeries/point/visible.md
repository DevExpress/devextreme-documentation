---
[note]When this option is set to **true**, some series points may still be hidden because of the [autoHidePointMarkers](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/autoHidePointMarkers.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/#autoHidePointMarkers') option.
