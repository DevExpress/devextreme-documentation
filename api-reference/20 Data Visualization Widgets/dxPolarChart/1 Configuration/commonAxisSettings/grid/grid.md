##### shortDescription
An object defining the configuration options for the [grid lines](/concepts/05%20Widgets/PolarChart/10%20Visual%20Elements/090%20Grid/01%20Grid.md '/Documentation/Guide/Widgets/PolarChart/Visual_Elements/#Grid') of an [axis](/concepts/05%20Widgets/PolarChart/10%20Visual%20Elements/050%20Axes/03%20Axes.md '/Documentation/Guide/Widgets/PolarChart/Visual_Elements/#Axes') in the **PolarChart** widget.

---
Grid lines are the reference lines used to improve the readability of a chart's visual data. The **grid** object exposes the properties that allow you to specify visibility and appearance settings for axis grid lines. To learn more about axis grid lines and their options, refer to the [Grid](/concepts/05%20Widgets/PolarChart/10%20Visual%20Elements/090%20Grid/01%20Grid.md '/Documentation/Guide/Widgets/PolarChart/Visual_Elements/#Grid') topic.
