default: 'hide'
acceptValues: 'hide' | 'ellipsis'

---
##### shortDescription
Decides whether those labels that overflow their tile/group should be hidden or truncated with ellipsis.

---
When configuring the widget using [ASP.NET MVC Wrappers](/concepts/35%20ASP.NET%20MVC%20Wrappers/20%20Fundamentals '/Documentation/Guide/ASP.NET_MVC_Wrappers/Fundamentals/'), you can specify this option using the `TreeMapResolveLabelOverflow` enum. This enum accepts the following values: `Hide` and `Ellipsis`.
