---
id: DataSource.Options.onLoadError
type: function(error)
---
---
##### shortDescription
A function that is executed when data loading fails.

##### param(error): Object
The error.

##### field(error.message): String
The error message.

---
---
##### jQuery

    <!--JavaScript-->
    var ds = new DevExpress.data.DataSource({
        onLoadError: function (error) {
            console.log(error.message);
        }
    });
