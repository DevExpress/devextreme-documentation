id: DataSource.group()
---
---
##### shortDescription
Gets the [group](/api-reference/30%20Data%20Layer/DataSource/1%20Configuration/group.md '/Documentation/ApiReference/Data_Layer/DataSource/Configuration/#group') property's value.

##### return: Object
A group expression.

---
---
##### jQuery

    <!--JavaScript-->
    var ds = new DevExpress.data.DataSource({
        // ...
        group: { selector: "employeeID", desc: true }
    });

    var groupExpr = ds.group(); // returns { selector: "employeeID", desc: true }
