##### shortDescription
Gets an object with current data processing settings.

##### return: Object
[Data processing settings](/api-reference/30%20Data%20Layer/CustomStore/LoadOptions '/Documentation/ApiReference/Data_Layer/CustomStore/LoadOptions/').
