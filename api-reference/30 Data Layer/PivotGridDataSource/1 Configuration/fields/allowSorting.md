---
id: PivotGridDataSource.Options.fields.allowSorting
type: Boolean
default: false
---
---
##### shortDescription
Specifies whether a user can change the field's sorting.

---
When this property is set to **true**, clicking the field's header in the [field chooser](/concepts/05%20UI%20Components/PivotGrid/010%20Visual%20Elements/10%20Field%20Chooser/01%20Field%20Chooser.md '/Documentation/Guide/UI_Components/PivotGrid/Visual_Elements/#Field_Chooser') and on the [field panel](/concepts/05%20UI%20Components/PivotGrid/010%20Visual%20Elements/15%20Field%20Panel.md '/Documentation/Guide/UI_Components/PivotGrid/Visual_Elements/#Field_Panel') changes the sorting direction. An arrow icon in the header indicates the current sorting direction.
