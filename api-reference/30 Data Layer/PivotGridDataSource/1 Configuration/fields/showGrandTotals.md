id: PivotGridDataSource.Options.fields.showGrandTotals
type: Boolean

---
##### shortDescription
Specifies whether to display the field's [grand totals](/concepts/05%20UI%20Components/PivotGrid/010%20Visual%20Elements/05%20Totals '/Documentation/Guide/UI_Components/PivotGrid/Visual_Elements/#Totals'). Applies only if the field is in the data [area](/api-reference/30%20Data%20Layer/PivotGridDataSource/1%20Configuration/fields/area.md '/Documentation/ApiReference/Data_Layer/PivotGridDataSource/Configuration/fields/#area').
