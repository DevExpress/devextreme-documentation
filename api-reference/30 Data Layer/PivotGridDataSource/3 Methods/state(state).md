##### shortDescription
Sets the **PivotGridDataSource** state. Part of the PivotGrid UI component's [state storing](/api-reference/10%20UI%20Components/dxPivotGrid/1%20Configuration/stateStoring '/Documentation/ApiReference/UI_Components/dxPivotGrid/Configuration/stateStoring/') feature.

##### param(state): Object
The state returned by the [state()](/api-reference/30%20Data%20Layer/PivotGridDataSource/3%20Methods/state().md '/Documentation/ApiReference/Data_Layer/PivotGridDataSource/Methods/#state') method. Pass **null** to reset the state to default.

---
