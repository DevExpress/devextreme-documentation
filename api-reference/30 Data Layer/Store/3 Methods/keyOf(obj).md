---
id: Store.keyOf(obj)
---
---
##### shortDescription
Gets a data item's key value.

##### return: any | String | Number
The data item's key value.

##### param(obj): Object
A data item.

---
---

    var key = store.keyOf({ id: 1, name: "John Doe" }); // returns 1
