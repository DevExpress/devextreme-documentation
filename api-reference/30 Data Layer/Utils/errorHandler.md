##### shortDescription
Specifies the function that is executed when a data layer object throws an error.

##### param(e): Error
A JavaScript **Error** object.

---
<!--->

    <!--JavaScript-->
    DevExpress.data.errorHandler = function (error) {
        console.log(error.message);
    };
