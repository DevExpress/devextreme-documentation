##### shortDescription
Creates a [Query](/api-reference/30%20Data%20Layer/Query '/Documentation/ApiReference/Data_Layer/Query/') instance.

##### return: Query
A **Query** instance.

##### param(array): Array<any>
Data to be associated with the **Query**.
