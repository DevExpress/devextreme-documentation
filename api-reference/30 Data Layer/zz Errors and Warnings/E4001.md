##### shortDescription
Occurs when the an unknown aggregating function is specified.

---
This error may occur in widgets that include the "summary" feature. For example, if you specify a not-valid [summaryType](/api-reference/10%20UI%20Widgets/dxDataGrid/1%20Configuration/summary/groupItems/summaryType.md '/Documentation/ApiReference/UI_Widgets/dxDataGrid/Configuration/summary/groupItems/#summaryType') for the **DataGrid** widget, this error will occur.
