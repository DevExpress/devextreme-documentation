##### shortDescription
Occurs when an unimplemented CustomStore method is called.

---
CustomStore requires that you implement its methods manually. To resolve the issue in question, implement the method specified in the error message you see in the browser console. See the [CustomStore API reference](/api-reference/30%20Data%20Layer/CustomStore/1%20Configuration '/Documentation/ApiReference/Data_Layer/CustomStore/Configuration/') for code examples.
