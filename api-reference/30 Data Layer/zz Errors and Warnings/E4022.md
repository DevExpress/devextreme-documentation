##### shortDescription
Occurs if [CustomStore](/api-reference/30%20Data%20Layer/CustomStore '/Documentation/ApiReference/Data_Layer/CustomStore/').[load](/api-reference/30%20Data%20Layer/CustomStore/1%20Configuration/load.md '/Documentation/ApiReference/Data_Layer/CustomStore/Configuration/#load') returns an `undefined` [LoadResultObject](/Documentation/ApiReference/Common_Types/data/custom-store/LoadResultObject/).**groupCount**.

---
This error is specific to the [CustomStore](/api-reference/30%20Data%20Layer/CustomStore '/Documentation/ApiReference/Data_Layer/CustomStore/') data store type. When **groupCount** is not in use, you can set it to `undefined` without this error.

The following code snippet demonstrates how to implement **CustomStore**.**load** to avoid this error:

---

##### jQuery

    <!-- tab: index.js -->
    var store = new DevExpress.data.CustomStore({
        load: function (loadOptions) {
            var deferred = $.Deferred();
            $.get(url, loadOptions)
                .done(function (response) {
                    deferred.resolve({
                        data: response.data,
                        groupCount: response.groupCount
                    });
                });
            return deferred.promise();
        }
    });

##### Angular

    <!-- tab: app.component.ts -->
    import { HttpClient } from '@angular/common/http';
    import { lastValueFrom } from 'rxjs';
    import { CustomStore } from 'devextreme-angular/common/data';
    import type { LoadOptions, LoadResult } from 'devextreme-angular/common/data';

    export class App {
        constructor(private http: HttpClient) {}

        dataSource = new CustomStore({
            load: async (options: LoadOptions): Promise<LoadResult> => {
                // ...

                const result = await lastValueFrom(this.http.get(url, loadParams));

                return {
                    data: result.data,
                    groupCount: result.groupCount,
                };
            },
        });
    }
