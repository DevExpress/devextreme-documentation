##### shortDescription
Configures visibility of the AI header menu button.

---
When this property is enabled, the AI column displays a header menu that includes the following items:

- "Autofill with AI"    
Opens the AI prompt editor.

- "Regenerate Data"    
Sends an AI request to regenerate data. 

- "Clear Data"    
Clears AI column cells.
