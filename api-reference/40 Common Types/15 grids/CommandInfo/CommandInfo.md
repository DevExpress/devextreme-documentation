##### shortDescription
Information about a predefined DataGrid/TreeList command.

---
**CommandInfo** contains the following fields:

- **name**: The predefined command's name ([PredefinedCommandNames](/api-reference/40%20Common%20Types/15%20grids/PredefinedCommandNames '{basewidgetpath}/PredefinedCommandNames/')).
- **args**: Command arguments. Refer to [PredefinedCommands](/api-reference/40%20Common%20Types/15%20grids/PredefinedCommands '{basewidgetpath}/PredefinedCommands/') for information about the arguments of each available command.
