##### shortDescription
A node in a filter expression tree ([FilterExprTree](/api-reference/40%20Common%20Types/15%20grids/FilterExprTree '{basewidgetpath}/FilterExprTree/')).

---
This object includes the following fields:

- **id**: A unique identifier for the node. Nodes reference each other by ID.
- **expr**: The filter expression ([FilterExpr](/api-reference/40%20Common%20Types/15%20grids/FilterExpr '{basewidgetpath}/FilterExpr/')).
