##### shortDescription
Configures the search panel.

##### field(highlightCaseSensitive): Boolean
Notifies the UI component whether search is case-sensitive to ensure that search results are highlighted correctly. Applies only if [highlightSearchText](/api-reference/10%20UI%20Components/GridBase/1%20Configuration/searchPanel/highlightSearchText.md '{basewidgetpath}/Configuration/searchPanel/#highlightSearchText') is **true**.

##### field(highlightSearchText): Boolean
Specifies whether found substrings should be highlighted.

##### field(placeholder): String
Specifies a placeholder for the search panel.

##### field(searchVisibleColumnsOnly): Boolean
Specifies whether the UI component should search against all columns or only visible ones.

##### field(text): String
Sets a search string for the search panel.

##### field(visible): Boolean
Specifies whether the search panel is visible or not.

##### field(width): String | Number
Specifies the width of the search panel in pixels.
