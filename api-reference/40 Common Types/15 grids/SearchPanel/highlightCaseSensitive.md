##### shortDescription
Notifies the UI component whether search is case-sensitive to ensure that search results are highlighted correctly. Applies only if [highlightSearchText](/api-reference/40%20Common%20Types/15%20grids/SearchPanel/highlightSearchText.md '{basewidgetpath}/Configuration/searchPanel/#highlightSearchText') is **true**.
