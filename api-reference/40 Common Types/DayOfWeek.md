##### shortDescription
A numeric union that identifies the day of the week.

---
The following values are accepted:

- `0`: Sunday
- `1`: Monday
- `2`: Tuesday
- `3`: Wednesday
- `4`: Thursday
- `5`: Friday
- `6`: Saturday
