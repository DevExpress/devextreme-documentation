##### shortDescription
The [native JavaScript event](https://developer.mozilla.org/en-US/docs/Web/Events) contained in the DevExtreme event.

---
DevExtreme events expose native JavaScript events with the **EventObject**.**originalEvent** field. Use this field to access native event methods and [properties](/api-reference/50%20Common/Object%20Structures/EventObject/2%20Fields '/Documentation/ApiReference/Common/Object_Structures/EventObject/Fields/') not passed to the parent DevExtreme event.
