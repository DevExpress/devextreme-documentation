---
id: EventObject.isDefaultPrevented()
---
---
##### shortDescription
Checks if the [preventDefault()](/api-reference/50%20Common/Object%20Structures/EventObject/3%20Methods/preventDefault().md '/Documentation/ApiReference/Common/Object_Structures/EventObject/Methods/#preventDefault') method was called on this event object.

##### return: Boolean
**true** if **preventDefault()** was called, **false** otherwise.
