##### shortDescription
Specifies the icon the FAB displays when the speed dial is closed.

---
#include uiwidgets-ref-icon

Use this property only if there are two or more [SpeedDialAction](/api-reference/10%20UI%20Components/dxSpeedDialAction '/Documentation/ApiReference/UI_Components/dxSpeedDialAction/') components on a page. Otherwise, specify the [icon](/api-reference/10%20UI%20Components/dxSpeedDialAction/1%20Configuration/icon.md '/Documentation/ApiReference/UI_Components/dxSpeedDialAction/Configuration/#icon') property in the SpeedDialAction.
