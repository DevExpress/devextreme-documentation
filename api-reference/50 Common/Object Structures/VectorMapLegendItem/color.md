##### shortDescription
The color of the legend item's marker.

---
If the legend is [grouped by color](/api-reference/10%20UI%20Components/dxVectorMap/1%20Configuration/legends/source/grouping.md '/Documentation/ApiReference/UI_Components/dxVectorMap/Configuration/legends/source/#grouping'), this property's value is different for each legend item. Otherwise, the value equals the [markerColor](/api-reference/10%20UI%20Components/dxVectorMap/1%20Configuration/legends/markerColor.md '/Documentation/ApiReference/UI_Components/dxVectorMap/Configuration/legends/#markerColor').
