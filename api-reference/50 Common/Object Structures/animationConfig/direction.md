##### shortDescription
Specifies the animation direction for the "slideIn" and "slideOut" animation types.

---
When using a widget as an <a href="https://docs.devexpress.com/DevExtremeAspNetMvc/400943/devextreme-aspnet-mvc-controls" target="_blank">ASP.NET MVC 5 Control</a> or a <a href="https://docs.devexpress.com/AspNetCore/400263/aspnet-core-controls#devextreme-based-aspnet-core-controls" target="_blank">DevExtreme-Based ASP.NET Core Control</a>, specify this option using the `Direction` enum. This enum accepts the following values: `Top`, `Bottom`, `Left` and `Right`.
