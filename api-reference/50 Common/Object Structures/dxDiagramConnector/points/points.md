##### shortDescription
Gets the connector's key points.

---
The **points** option contains an array of connector points. Each point is specified by its x- and y-coordinates in [units](/api-reference/10%20UI%20Widgets/dxDiagram/1%20Configuration/units.md '/Documentation/ApiReference/UI_Widgets/dxDiagram/Configuration/#units').

The **points** option value has the following format: ```[{"x":264,"y":384},{"x":288,"y":420},{"x":480,"y":420}]```.
