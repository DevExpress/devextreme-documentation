##### shortDescription
The new shape position (x- and y-coordinates) specified in [units](/api-reference/10%20UI%20Widgets/dxDiagram/1%20Configuration/units.md '/Documentation/ApiReference/UI_Widgets/dxDiagram/Configuration/#units').

---
The **newPosition** option contains the shape coordinates in the following format: ```{x:204,y:48}```.
