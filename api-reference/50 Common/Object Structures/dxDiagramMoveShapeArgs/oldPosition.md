##### shortDescription
The previous shape position (x- and y-coordinates) specified in [units](/api-reference/10%20UI%20Components/dxDiagram/1%20Configuration/units.md '/Documentation/ApiReference/UI_Components/dxDiagram/Configuration/#units').

---
The **oldPosition** property contains the shape coordinates in the following format: ```{x:154,y:70}```.
