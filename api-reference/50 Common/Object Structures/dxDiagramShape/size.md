##### shortDescription
Specifies the shape size in in [units](/api-reference/10%20UI%20Widgets/dxDiagram/1%20Configuration/units.md '/Documentation/ApiReference/UI_Widgets/dxDiagram/Configuration/#units').

---
The **size** option contains the shape size in the following format: ```{"width":120,"height":96}```.
