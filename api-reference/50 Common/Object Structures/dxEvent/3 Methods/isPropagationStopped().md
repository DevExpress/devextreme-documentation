##### shortDescription
Checks if the [stopPropagation()](/api-reference/50%20Common/Object%20Structures/EventObject/3%20Methods/stopPropagation().md '/Documentation/ApiReference/Common/Object_Structures/EventObject/Methods/#stopPropagation') method was called on this event object.

##### return: Boolean
**true** if **stopPropagation()** was called, **false** otherwise.
