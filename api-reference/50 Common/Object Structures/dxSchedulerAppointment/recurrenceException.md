##### shortDescription
Specifies the start date and time of one or more appointments to exclude from a series. This option requires that you also set [recurrenceRule](/api-reference/50%20Common/Object%20Structures/dxSchedulerAppointment/recurrenceRule.md '/Documentation/ApiReference/Common/Object_Structures/dxSchedulerAppointment/#recurrenceRule').
