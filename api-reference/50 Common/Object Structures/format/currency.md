##### shortDescription
Specifies a 3-letter <a href="http://www.currency-iso.org/en/home/tables/table-a1.html" target="_blank">ISO 4217</a> code for currency. Applies only if the [type](/api-reference/50%20Common/Object%20Structures/format/type.md '/Documentation/ApiReference/Common/Object_Structures/format/#type') is *"currency"*.
