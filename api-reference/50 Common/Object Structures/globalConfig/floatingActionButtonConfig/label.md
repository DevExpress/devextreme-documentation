##### shortDescription
Specifies the text label displayed inside the FAB.

---
Use this property only if there are two or more [SpeedDialAction](/api-reference/10%20UI%20Widgets/dxSpeedDialAction '/Documentation/ApiReference/UI_Components/dxSpeedDialAction/') components on a page. Otherwise, specify the [label](/api-reference/10%20UI%20Widgets/dxSpeedDialAction/1%20Configuration/label.md '/Documentation/ApiReference/UI_Components/dxSpeedDialAction/Configuration/#label') property in the SpeedDialAction.
