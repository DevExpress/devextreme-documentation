##### shortDescription
Specifies the text label displayed inside the FAB.

---
Use this option only if there are two or more [SpeedDialAction](/api-reference/10%20UI%20Widgets/dxSpeedDialAction '/Documentation/ApiReference/UI_Widgets/dxSpeedDialAction/') components on a page. Otherwise, specify the [label](/api-reference/10%20UI%20Widgets/dxSpeedDialAction/1%20Configuration/label.md '/Documentation/ApiReference/UI_Widgets/dxSpeedDialAction/Configuration/#label') option in the **SpeedDialAction**.
