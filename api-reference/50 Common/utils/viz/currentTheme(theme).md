##### shortDescription
Changes the current theme for all data visualization UI components on the page.

##### param(theme): String
The new theme's name.

---
Only the UI components created after calling this method use the new theme. Use the [DevExpress.viz.refreshTheme()](/api-reference/50%20Common/utils/viz/refreshTheme().md '/Documentation/ApiReference/Common/utils/viz/#refreshTheme') method to refresh the styling settings in other widgets.
