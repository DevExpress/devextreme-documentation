##### shortDescription
Refreshes the current theme and palette in all data visualization UI components on the page.

---
The **refreshTheme()** method calls [refresh()](/api-reference/10%20UI%20Components/BaseChart/3%20Methods/refresh().md '/Documentation/ApiReference/UI_Components/dxChart/Methods/#refresh') to reload data and update the UI of all data visualization components after applying a new theme.
