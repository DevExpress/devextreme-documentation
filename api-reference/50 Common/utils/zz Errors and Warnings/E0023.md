##### shortDescription
Occurs when a name is not specified for a [dxTemplate](/api-reference/10%20UI%20Widgets/Markup%20Components/dxTemplate '/Documentation/ApiReference/UI_Components/Markup_Components/dxTemplate/') markup component.

---
Specify a name using the **name** configuration property of the dxTemplate component.
