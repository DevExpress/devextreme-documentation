##### shortDescription
Occurs when either the **min** or **max** property is not specified for the ['range'](/api-reference/10%20UI%20Widgets/dxValidator/8%20Validation%20Rules/RangeRule '/Documentation/ApiReference/UI_Components/dxValidator/Validation_Rules/RangeRule/') rule.

---
The value of the **min**/**max** property must not be **null** or **undefined**.
