##### shortDescription
Occurs if an [AsyncRule](/api-reference/10%20UI%20Components/dxValidator/8%20Validation%20Rules/AsyncRule '/Documentation/ApiReference/UI_Components/dxValidator/Validation_Rules/AsyncRule/')'s [validationCallback](/api-reference/10%20UI%20Components/dxValidator/8%20Validation%20Rules/AsyncRule '/Documentation/ApiReference/UI_Components/dxValidator/Validation_Rules/AsyncRule/'#validationCallback) does not return a promise.

---
Return a <a href="https://api.jquery.com/Types/#Promise" target="_blank">jQuery</a> or <a href="https://api.jquery.com/Types/#Promise" target="_blank">native promise</a> from the **validationCallback**.
