##### shortDescription
Appears when the DevExtreme runtime key in the application configuration is missing, invalid, or not recognized.

---
To resolve this warning:

1. Obtain your DevExpress license key from the [DevExpress Download Manager](https://www.devexpress.com/ClientCenter/DownloadManager/) and register it on your machine.
2. Run the `devextreme-license` CLI tool to generate a DevExtreme-compatible runtime key.
3. Specify the generated key in [GlobalConfig](/api-reference/50%20Common/Object%20Structures/GlobalConfig '/Documentation/ApiReference/Common/Object_Structures/GlobalConfig/').

For more information, refer to the following article: [Licensing](/concepts/Common/Licensing/00%20Licensing.md '/Documentation/Guide/Common/Licensing/').
