##### shortDescription
Appears when the DevExtreme runtime key does not cover the version of DevExtreme used within a project.

---
To resolve this warning, you can:

- Use the DevExtreme version linked to your DevExpress license key (the DevExpress [Download Manager](https://www.devexpress.com/ClientCenter/DownloadManager/) displays a list of licensed versions).

- [Renew your DevExpress Subscription](https://www.devexpress.com/buy/renew/). Once renewed, download and re-register your updated DevExpress license key, then run the `devextreme-license` CLI tool to regenerate the runtime key.

For additional information, refer to the following help topic: [Registered License Key Is Outdated](https://docs.devexpress.com/GeneralInformation/405494/trial-register/set-up-your-dev-express-license-key#version-errors---registered-license-key-is-outdated).
