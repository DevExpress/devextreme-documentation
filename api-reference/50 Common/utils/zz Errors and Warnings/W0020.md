##### shortDescription
Occurs when the license key is expired.

---
The license key does not match the DevExtreme version.

To proceed, you can:

- use the DevExtreme version linked to your license key (see [Download Manager](https://www.devexpress.com/ClientCenter/DownloadManager/))

- [renew your DevExpress Subscription](https://www.devexpress.com/buy/renew/) (once you renew your subscription, you can get product updates and support)

If you have licensing-related questions or need help with a renewal, please submit a support ticket via the [DevExpress Support Center](https://supportcenter.devexpress.com/ticket/create).
