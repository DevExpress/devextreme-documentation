##### shortDescription
Appears when the DevExtreme runtime key verification fails.

---
To resolve this warning, run the `devextreme-license` CLI tool to regenerate the DevExtreme runtime key and specify the newly generated key in [GlobalConfig](/api-reference/50%20Common/Object%20Structures/GlobalConfig '/Documentation/ApiReference/Common/Object_Structures/GlobalConfig/'). If the warning persists, ensure that your DevExpress license key is registered correctly on your machine, then run `devextreme-license` again.

If you have a valid DevExpress license key and this problem persists, submit a support ticket in the [DevExpress Support Center](https://supportcenter.devexpress.com/ticket/create).

For additional information, refer to the following help topic: [License Key for DevExpress Products](https://docs.devexpress.com/GeneralInformation/405494/trial-register/set-up-your-dev-express-license-key#manual-registration-additional-step-for-devextreme-jsts-developers).
