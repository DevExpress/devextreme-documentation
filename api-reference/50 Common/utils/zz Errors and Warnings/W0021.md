##### shortDescription
Appears when the license key verification fails.

---
To avoid this error, please specify the correct DevExtreme license key in `GlobalConfig`. If you continue to encounter the error, visit the DevExpress [Download Manager](https://www.devexpress.com/ClientCenter/DownloadManager/) to obtain a valid key.

If you have a valid license key and this problem persists, please submit a support ticket via the [DevExpress Support Center](https://supportcenter.devexpress.com/ticket/create).
