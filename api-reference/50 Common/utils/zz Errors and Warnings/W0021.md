##### shortDescription
Occurs when the license key verification fails.

---
Make sure you specify the correct key in the `GlobalConfig`. If you still get an error, use the [Download Manager](https://www.devexpress.com/ClientCenter/DownloadManager/) to obtain a valid key.

If you have a valid license and this problem persists, please submit a support ticket via the [DevExpress Support Center](https://supportcenter.devexpress.com/ticket/create).
