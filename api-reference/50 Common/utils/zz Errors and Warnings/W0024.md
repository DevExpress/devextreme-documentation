##### shortDescription
Appears if the DevExtreme .NET license key is entered instead of a DevExtreme key.

---
To remove this warning, enter the correct DevExtreme (React, Angular, Vue, JS) license key.
 
For more information on DevExtreme licensing-related questions, refer to the following help topic: [Licensing](/concepts/Common/Licensing/00%20Licensing.md '/Documentation/Guide/Common/Licensing/').
