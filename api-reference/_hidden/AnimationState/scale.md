---
##### shortDescription
A value that controls element size.

---
The following example shows how to use this property:

    <!-- tab: JavaScript -->
    // Hide the element
    scale: 0,

    // Display the element in its original size (100%)
    scale: 1,

    // Scale the element up to 150%
    scale: 1.5
