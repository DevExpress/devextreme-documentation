##### shortDescription
Configures the image to be displayed in the annotation. Applies only if the [type](/api-reference/_hidden/BaseChartAnnotationConfig/type.md '{basewidgetpath}/Configuration/annotations/#type') is *"image"*.

---
To display an image, assign its URL to the **url** option and use the **height** and **width** options to resize the image. Otherwise, assign the URL directly to the **image** option.
