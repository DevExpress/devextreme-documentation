##### shortDescription
[tags] ctp

Zero-based index of a page into which a new row should be inserted. Applies only if the [type]({currentpath}/#type) is *"insert"* and the [pager]({basewidgetpath}/Configuration/pager/) is used.

---

#include common-ctp-note-wo-devextreme 

When not specified, this field equals the [paging]({basewidgetpath}/Configuration/paging/).[pageIndex]({basewidgetpath}/Configuration/paging/#pageIndex) value. To add a row to the last page, set this field to -1.
