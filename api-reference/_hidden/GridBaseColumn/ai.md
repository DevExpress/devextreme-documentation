##### shortDescription
Configures AI options for the column.

---
Configure this property to integrate an AI column.

[note]

- To ensure the AI column functions correctly, specify key values in the {WidgetName} data source.
- If {WidgetName} automatically hides an AI column (when [columnHidingEnabled](/api-reference/10%20UI%20Components/GridBase/1%20Configuration/columnHidingEnabled.md '{basewidgetpath}/Configuration/#columnHidingEnabled') is `true`), users cannot access the [AI header menu](/api-reference/40%20Common%20Types/15%20grids/ColumnAIOptions/showHeaderMenu.md '{basewidgetpath}/Configuration/columns/ai/#showHeaderMenu').

[/note]
