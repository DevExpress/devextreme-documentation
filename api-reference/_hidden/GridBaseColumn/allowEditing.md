id: GridBaseColumn.allowEditing
type: Boolean

---
##### shortDescription
Specifies whether a user can edit values in the column at runtime. By default, inherits the value of the **editing**.[allowUpdating](/api-reference/10%20UI%20Widgets/dxDataGrid/1%20Configuration/editing/allowUpdating.md '{basewidgetpath}/Configuration/editing/#allowUpdating') option.

---
[note] If values in the column are calculated customarily using the [calculateCellValue](/api-reference/_hidden/GridBaseColumn/calculateCellValue.md '{basewidgetpath}/Configuration/columns/#calculateCellValue') option, they cannot be edited at runtime.
