##### shortDescription
Specifies a caption for the column.

---
Use this option to display a descriptive or friendly name for the column. If this option is not set, the caption will be generated from the name of the [dataField](/api-reference/_hidden/GridBaseColumn/dataField.md '{basewidgetpath}/Configuration/columns/#dataField').
