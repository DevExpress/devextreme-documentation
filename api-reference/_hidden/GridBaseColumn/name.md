##### shortDescription
Specifies a unique identifier of the column. The value is inherited from the [dataField]({basewidgetpath}/Configuration/columns/#dataField) by default.

---

This option's value is used to refer to the column in your code, for example, when [changing a column option]({basewidgetpath}/Methods/#columnOptionid_optionName_optionValue).
