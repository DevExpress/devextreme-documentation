##### shortDescription
Specifies a unique identifier of the column. 

---

This value is used to refer to the column in your code, for example, when [changing a column option]({basewidgetpath}/Methods/#columnOptionid_optionName_optionValue). It is also used in the editColumnName option to get the name of a column being edited. If you want to use a value different from the autogenerated one, specify it with this option.
