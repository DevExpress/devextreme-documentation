##### shortDescription
Specifies the column's unique identifier. If not set in code, this value is inherited from the [dataField]({basewidgetpath}/Configuration/columns/#dataField).

---

This option's value is used to refer to the column in code, for example, when [changing a column option]({basewidgetpath}/Methods/#columnOptionid_optionName_optionValue).
