##### shortDescription
Specify the name of the used font. Default font names are listed below:

* Courier
* Courier-Bold
* Courier-BoldOblique
* Courier-Oblique
* Helvetica
* Helvetica-Bold
* Helvetica-BoldOblique
* Helvetica-Oblique
* Symbol
* Times-Roman
* Times-Bold
* Time-Italic
* Time-BoldItalic
