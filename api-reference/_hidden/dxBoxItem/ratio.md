##### shortDescription
Specifies the ratio value used to count the item element size along the main direction.

---
We do not recommend changing this option at runtime. If you want to create a responsive layout in your application, use [ResponsiveBox](/concepts/05%20UI%20Components/ResponsiveBox/00%20Overview.md '/Documentation/Guide/UI_Components/ResponsiveBox/Overview/') instead.
