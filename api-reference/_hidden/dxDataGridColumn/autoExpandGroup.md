---
id: dxDataGridColumn.autoExpandGroup
type: Boolean
default: true
---
---
##### shortDescription
Specifies whether groups appear expanded or not when records are grouped by a specific column. Setting this property makes sense only when grouping is [allowed](/api-reference/_hidden/dxDataGridColumn/allowGrouping.md '/Documentation/ApiReference/UI_Components/dxDataGrid/Configuration/columns/#allowGrouping') for this column.
