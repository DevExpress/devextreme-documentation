##### shortDescription
Specifies the data field's caption.

---
If you do not specify this option, the **FilterBuilder** generates the caption based on the [dataField](/api-reference/_hidden/dxFilterBuilderField/dataField.md '/Documentation/ApiReference/UI_Widgets/dxFilterBuilder/Configuration/fields/#dataField') option. For example, if **dataField** is *"fullName"*, the caption is *"Full Name"*.
