##### shortDescription
Specifies text for the "reset" operation.

---

The "reset" operation restores the default filter, which depends on a column's data type. Use the **column**.[selectedFilterOperation](/Documentation/ApiReference/UI_Components/dxGantt/Configuration/columns/#selectedFilterOperation) property to specify the default filter operation for a column's filter row.
