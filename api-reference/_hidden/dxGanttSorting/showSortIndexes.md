---
##### shortDescription
Specifies whether to display sort indexes in column headers. Applies only when **sorting**.[mode](/api-reference/_hidden/dxGanttSorting/mode.md '/Documentation/ApiReference/UI_Components/dxGantt/Configuration/sorting/#mode') is "multiple" and data is sorted by two or more columns.
