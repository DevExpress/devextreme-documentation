##### shortDescription
Specifies values for a format with multiple choices. Should be used with the [name](/api-reference/_hidden/dxHtmlEditorToolbarItem/name.md).

---
Formats with multiple choices are represented by [SelectBox](/api-reference/10%20UI%20Widgets/dxSelectBox) UI components whose [properties](/api-reference/10%20UI%20Widgets/dxSelectBox/1%20Configuration) you can specify in the [options](/api-reference/_hidden/dxHtmlEditorToolbar/items/options.md '/Documentation/ApiReference/UI_Components/dxHtmlEditor/Configuration/toolbar/items/#options') object.
