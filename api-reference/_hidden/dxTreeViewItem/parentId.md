##### shortDescription
Holds the key of the parent item.

---
If you did not use the [rootValue](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#rootValue) property to set a custom id for the root item, you can omit this field for the top-level item objects or set it to 0 (default).

Specify this field regardless of the item template type (default or [custom](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#itemTemplate)).
