A user can use the following keys to interact with the ActionSheet component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>Tab</td>
        <td>Moves focus to the next button.</td>
    </tr>
    <tr>
        <td>Enter or Space</td>
        <td>Identical to a click.</td>
    </tr>
</table>
