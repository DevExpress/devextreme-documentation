An end user can use the following keys to interact with the Autocomplete component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>&uarr; / &darr;</td>
        <td>Moves focus to the previous/following menu item.</td>
    </tr>
    <tr>
        <td>Page Up / Page Down</td>
        <td>Moves focus to the first/last menu item.</td>
    </tr>
    <tr>
        <td>Enter</td>
        <td>Selects focused menu item.</td>
    </tr>
    <tr>
        <td>Esc</td>
        <td>Closes the drop-down menu.</td>
    </tr>
</table>
