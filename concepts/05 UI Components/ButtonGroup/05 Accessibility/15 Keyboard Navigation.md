A user can use the following keys to interact with the ButtonGroup component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>&larr; &uarr; or PageUp</td>
        <td>Moves focus to the previous button.</td>
    </tr>
    <tr>
        <td>&rarr; &darr; or PageDown</td>
        <td> Moves focus to the next button.</td>
    </tr>
    <tr>
        <td>Home</td>
        <td>Moves focus to the first button.</td>
    </tr>
    <tr>
        <td>End</td>
        <td>Moves focus to the last button.</td>
    </tr>
    <tr>
        <td>Enter or Space</td>
        <td>Identical to a click.</td>
    </tr>
</table>
