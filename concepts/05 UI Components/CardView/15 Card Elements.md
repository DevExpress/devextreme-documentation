## Key Features

- **Multi-Functional Header**    
CardView cards feature a [header](/Documentation/ApiReference/UI_Components/dxCardView/Configuration/cardHeader/) that displays multiple controls, such as [selection](/Documentation/ApiReference/UI_Components/dxCardView/Configuration/selection/) checkboxes and [editing](/Documentation/ApiReference/UI_Components/dxCardView/Configuration/editing/) command buttons. You can add custom controls in the **cardHeader**.[items[]](/Documentation/ApiReference/UI_Components/dxCardView/Configuration/cardHeader/items/) array.
- **Images in Cards**    
Configure the [cardCover](/Documentation/ApiReference/UI_Components/dxCardView/Configuration/cardCover/) property to display images in cards.
- **Customizable Fields**    
Customize how cards display data with [columns[]](/Documentation/ApiReference/UI_Components/dxCardView/Configuration/columns/) properties such as [caption](/Documentation/ApiReference/UI_Components/dxCardView/Configuration/columns/#caption), [customizeText](/Documentation/ApiReference/UI_Components/dxCardView/Configuration/columns/#customizeText), and [fieldTemplate](/Documentation/ApiReference/UI_Components/dxCardView/Configuration/columns/#fieldTemplate).
- **Card Footers**    
Configure a custom card footer with the [cardFooterTemplate](/Documentation/ApiReference/UI_Components/dxCardView/Configuration/#cardFooterTemplate) property.
