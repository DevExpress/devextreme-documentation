A user can use the following keys to interact with the Chat component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>Enter</td>
        <td>Enters a message.<br>Applies change to the edited message.</td>
    </tr>
    <tr>
        <td>Esc</td>
        <td>Cancels message editing.</td>
    </tr>
</table>

[note] Keyboard navigation is not supported in the message list.
