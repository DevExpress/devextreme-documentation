Create a [Dialogflow Agent](https://dialogflow.cloud.google.com/#/newAgent) and [configure it](https://docs.cloud.google.com/dialogflow/es/docs/intents-overview) as your needs dictate. 

Then, obtain a Google Cloud key:

1. Navigate to Google Cloud Console → IAM & Admin → [Service Accounts](https://console.cloud.google.com/iam-admin/serviceaccounts). 
2. Create a new or use an existing service account. 
3. Create a key (JSON) for this account. 
4. Save the resulting JSON file as `key.json`.

In the Google Cloud Console, find your [Project ID](https://console.cloud.google.com/home/dashboard).
