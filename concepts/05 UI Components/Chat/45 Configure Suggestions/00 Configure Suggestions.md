DevExtreme Chat can display suggestion buttons (a DevExtreme [ButtonGroup](/Documentation/Guide/UI_Components/ButtonGroup/Getting_Started_with_ButtonGroup/) component) above the message input field. To activate this capability, use the [suggestions](/Documentation/ApiReference/UI_Components/dxChat/Configuration/#suggestions) object.

To display suggestion buttons, specify the **suggestions**.**items[]** array. Use ButtonGroup [item fields](/Documentation/ApiReference/UI_Components/dxButtonGroup/Configuration/items/) to configure the appearance and visual elements of individual buttons. You can also specify custom fields to implement additional functionality. For instance, you can store predefined messages in a custom field and use short labels (**items[]**.[text](/Documentation/ApiReference/UI_Components/dxButtonGroup/Configuration/items/#text) values).
