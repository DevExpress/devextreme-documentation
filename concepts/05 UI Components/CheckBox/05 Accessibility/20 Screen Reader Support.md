The CheckBox component supports screen readers and complies to <a href="https://www.w3.org/WAI/standards-guidelines/aria/" target="_blank">WAI-ARIA</a> standards.

The CheckBox's aria label depends on the properties you use. If you specify the [text](/api-reference/10%20UI%20Components/dxCheckBox/1%20Configuration/text.md '//Documentation/ApiReference/UI_Components/dxCheckBox/Configuration/#text') or [hint](/api-reference/10%20UI%20Components/Widget/1%20Configuration/hint.md '/Documentation/ApiReference/UI_Components/dxCheckBox/Configuration/#hint') property, its value is copied to the aria-label. In other cases, use [elementAttr](/api-reference/10%20UI%20Components/DOMComponent/1%20Configuration/elementAttr.md '/Documentation/ApiReference/UI_Components/dxCheckBox/Configuration/#elementAttr') to specify the aria-label directly.
