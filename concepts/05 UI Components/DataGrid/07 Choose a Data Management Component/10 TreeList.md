Displays hierarchical data in a tree with grid-like features. Suitable for parent-child structures. For an expanded list of features, see [TreeList Key Features](/concepts/05%20UI%20Components/TreeList/02%20Overview/10%20Key%20Features.md '/Documentation/Guide/UI_Components/TreeList/Overview/#Key_Features').

<img src="/images/treelist/tree-list-choose.png" alt="DevExtreme TreeList" />

### Typical Usage Scenarios

- Organizational structures: Customer Relationship Management (CRM) and Human Resource Management (HRM)
- Category hierarchies (for example, project tasks) 
- File structures  

[note] Use the [TreeList](/concepts/05%20UI%20Components/TreeList/02%20Overview/00%20Overview.md '/Documentation/Guide/UI_Components/TreeList/Overview/') for hierarchical (parent-child) structures, or when both tree and table formats are needed.
