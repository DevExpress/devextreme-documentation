Do not enable [columnAutoWidth](/api-reference/10%20UI%20Components/GridBase/1%20Configuration/columnAutoWidth.md '/Documentation/ApiReference/UI_Components/dxDataGrid/Configuration/#columnAutoWidth') property. To further optimize rendering, set a fixed width for all columns. To do this, use a column's [width](/api-reference/_hidden/GridBaseColumn/width.md '/Documentation/ApiReference/UI_Components/dxDataGrid/Configuration/columns/#width') or a component's [columnWidth](/api-reference/10%20UI%20Components/GridBase/1%20Configuration/columnWidth.md '/Documentation/ApiReference/UI_Components/dxDataGrid/Configuration/#columnWidth') properties to specify the width for each column or all columns, respectively. Note that you should use individual settings because command columns ignore **columnWidth**.
