The DateRangeBox component supports screen readers and complies to <a href="https://www.w3.org/WAI/standards-guidelines/aria/" target="_blank">WAI-ARIA</a> standards. 

To improve accessibility, use the [startDateInputAttr](/Documentation/ApiReference/UI_Components/dxDateRangeBox/Configuration/#startDateInputAttr) and [endDateInputAttr](/Documentation/ApiReference/UI_Components/dxDateRangeBox/Configuration/#endDateInputAttr) properties to specify <a href="https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-label" target="_blank">aria-label</a> for the component. You do not need to specify `aria-label` if you use [startDateLabel](/Documentation/ApiReference/UI_Components/dxDateRangeBox/Configuration/#startDateLabel) or [endDateLabel](/Documentation/ApiReference/UI_Components/dxDateRangeBox/Configuration/#endDateLabel) correspondingly.
