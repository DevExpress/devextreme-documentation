This example specifies the following Draggable properties:

- [boundary](/Documentation/ApiReference/UI_Components/dxDraggable/Configuration/#boundary): Constrains Draggable movement inside a container (`.board`).
- [group](/Documentation/ApiReference/UI_Components/dxDraggable/Configuration/#group): Specifies an interaction group (used in events such as [onDragMove](/Documentation/ApiReference/UI_Components/dxDraggable/Configuration/#onDragMove)).
