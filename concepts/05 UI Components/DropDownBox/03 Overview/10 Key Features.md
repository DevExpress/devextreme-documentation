- **Input Field Customization**    
You can customize the DropDownBox input field as your needs dictate. Specify the following properties to implement your customizations:
    - [label](/Documentation/ApiReference/UI_Components/dxDropDownBox/Configuration/#label) and [labelMode](/Documentation/ApiReference/UI_Components/dxDropDownBox/Configuration/#labelMode)    
    Configure DropDownBox labels.
    - [placeholder](/Documentation/ApiReference/UI_Components/dxDropDownBox/Configuration/#placeholder)    
    Specifies a placeholder value.
    - [showClearButton](/Documentation/ApiReference/UI_Components/dxDropDownBox/Configuration/#showClearButton) and [showDropDownButton](/Documentation/ApiReference/UI_Components/dxDropDownBox/Configuration/#showClearButton)    
    Specify visibility of the predefined drop-down and clear buttons.
    - [buttons[]](/Documentation/ApiReference/UI_Components/dxDropDownBox/Configuration/buttons/)    
    Configure custom DropDownBox input field buttons.
