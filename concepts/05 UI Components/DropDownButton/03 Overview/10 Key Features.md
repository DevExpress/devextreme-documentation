- **Action Binding**    
DropDownButton can trigger actions on button click ([onButtonClick](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/onButtonClick.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#onButtonClick')) or list item click ([onItemClick](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/onItemClick.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#onItemClick')). You can also specify individual [click handlers](/api-reference/_hidden/dxDropDownButtonItem/onClick.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/items/#onClick') for each item within the component data source.

- **Appearance Settings**    
You can specify an [icon](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/icon.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#icon') and [text](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/text.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#text'), as well as configure [drop-down arrow](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/showArrowIcon.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#showArrowIcon') visibility to customize component appearance.

- **Item Configuration**    
You can configure [text](/api-reference/_hidden/CollectionWidgetItem/text.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/items/#text'), [icons](/api-reference/_hidden/dxListItem/icon.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/items/#icon'), and [badges](/api-reference/_hidden/dxListItem/badge.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/items/#badge') to customize drop-down item appearance. The component also allows you to implement unique [click handlers](/api-reference/_hidden/dxDropDownButtonItem/onClick.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/items/#onClick') for each item, as well as [custom templates](/api-reference/_hidden/CollectionWidgetItem/template.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/items/#template').

- **Split Button Support**    
DevExtreme DropDownButton supports a [split button](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/splitButton.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#splitButton') mode. In this mode, the component separates the action and drop-down buttons.

- **Predefined Styling Modes**    
You can choose one of three [styling modes](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/stylingMode.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#stylingMode') to configure DropDownButton appearance: text only, contained, and outlined (default). The component also includes [four built-in colors](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/type.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#type').

- **Selection**    
When [selection](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/useSelectMode.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#useSelectMode') is enabled, the component displays the selected item text and icon. You can [handle selection change](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/onSelectionChanged.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#onSelectionChanged') and process [selected item](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/selectedItemKey.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#selectedItemKey') data at runtime.

- **Accessibility and Keyboard Navigation**    
DropDownButton [conforms](/concepts/05%20UI%20Components/DropDownButton/05%20Accessibility/00%20Accessibility.md '/Documentation/Guide/UI_Components/DropDownButton/Accessibility/') to WCAG 2.x, European Accessibility Act (EAA), and Americans with Disabilities Act (ADA) standards. The component supports keyboard navigation and custom key handlers. TagBox also ships with right-to-left ([RTL](/api-reference/10%20UI%20Components/DOMComponent/1%20Configuration/rtlEnabled.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#rtlEnabled')) representation support.

- **Custom Templates**    
DropDownButton supports custom markup for the component [action button](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/template.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#template') and the [drop-down content](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/dropDownContentTemplate.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#dropDownContentTemplate').  You can also specify custom templates for [specific](/api-reference/_hidden/CollectionWidgetItem/template.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/items/#template') drop-down items or a [common template](/api-reference/10%20UI%20Components/dxDropDownButton/1%20Configuration/itemTemplate.md '/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#itemTemplate') for all items.
