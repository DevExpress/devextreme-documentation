A user can use the following keys to interact with the DropDownButton component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>Left Arrow</td>
        <td>Moves focus to the action button when <a href="/Documentation/ApiReference/UI_Components/dxDropDownButton/Configuration/#splitButton">splitButton</a> is enabled.</td>
    </tr>
    <tr>
        <td>Right Arrow</td>
        <td>Moves focus to the toggle button when <b>splitButton</b> is enabled.</td>
    </tr>
    <tr>
        <td rowspan="2">Enter / Space</td>
        <td>Identical to a click.</td>
    </tr>
    <tr>
        <td>Selects a menu item when focused.</td>
    </tr>
    <tr>
        <td>Up Arrow / Down Arrow/td>
        <td>Moves focus to the previous/following menu item.</td>
    </tr>
    <tr>
        <td>Down Arrow</td>
        <td>Opens the drop-down menu.</td>
    </tr>
    <tr>
        <td>Esc</td>
        <td>Closes the drop-down menu.</td>
    </tr>
    <tr>
        <td>Page Up / Page Down</td>
        <td>Moves focus to the first/last menu item on the page.</td>
    </tr>
    <tr>
        <td>Home / End</td>
        <td>Moves focus to the first/last menu item.</td>
    </tr>
</table>
