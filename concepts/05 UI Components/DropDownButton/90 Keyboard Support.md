An end user can use the following keys to interact with the UI component.

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>&larr;</td>
        <td>Moves focus to the action button when split button is enabled.</td>
    </tr>
    <tr>
        <td>&rarr;</td>
        <td>Moves focus to the toggle button when split button is enabled.</td>
    </tr>
    <tr>
        <td rowspan="3">Enter / Space</td>
        <td>Opens the popup when the toggle button is focused.</td>
    </tr>
    <tr>
        <td>Simulates a click action when the action button is focused.</td>
    </tr>
    <tr>
        <td>Selects list item when focused.</td>
    </tr>
    <tr>
        <td>&uarr; / &darr;</td>
        <td>Moves focus to the previous/following item.</td>
    </tr>
    <tr>
        <td>&darr;</td>
        <td>Opens the dropdown menu.</td>
    </tr>
    <tr>
        <td>Esc</td>
        <td>Closes the dropdown menu.</td>
    </tr>
    <tr>
        <td>Page Up / Page Down</td>
        <td>Moves focus to the first/last item of the list page.</td>
    </tr>
    <tr>
        <td>Home / End</td>
        <td>Moves focus to the first/last list item.</td>
    </tr>
</table>
