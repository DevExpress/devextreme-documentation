- **Validation**    
DevExtreme includes a [validation engine](/Documentation/Guide/UI_Components/Form/Getting_Started_with_Form/#Validate_the_Form_Data) that checks edited Form values before they are saved.

- **Flexible Layout**    
Form can organize items into [columns](/Documentation/Guide/UI_Components/Form/Organize_Simple_Items/In_Columns/), [groups](/Documentation/Guide/UI_Components/Form/Organize_Simple_Items/In_Groups/), and [tabs](/Documentation/Guide/UI_Components/Form/Organize_Simple_Items/In_Tabs/).

- **Adaptability**    
You can [specify](/Documentation/Guide/UI_Components/Form/Organize_Simple_Items/In_Columns/#Layout_Depending_on_the_Screen_Width) different Form layouts for different screen widths.

- **Extensive Editor Selection**    
Choose from [18 DevExtreme editors](/Documentation/ApiReference/UI_Components/dxForm/Types/#FormItemComponent) to display Form data.
