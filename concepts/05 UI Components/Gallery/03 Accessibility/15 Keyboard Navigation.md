A user can use the following keys to interact with the Gallery component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>&larr; / &uarr;</td>
        <td>Shows the previous image.</td>
    </tr>
    <tr>
        <td>&rarr; / &darr;</td>
        <td>Shows the following image.</td>
    </tr>
</table>
