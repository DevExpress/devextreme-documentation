The Gallery component supports screen readers and complies to <a href="https://www.w3.org/WAI/standards-guidelines/aria/" target="_blank">WAI-ARIA</a> standards.

To improve accessibility, specify the [imageAlt](/api-reference/_hidden/dxGalleryItem/imageAlt.md '/Documentation/ApiReference/UI_Components/dxGallery/Configuration/items/#imageAlt') attribute for each Gallery image.
