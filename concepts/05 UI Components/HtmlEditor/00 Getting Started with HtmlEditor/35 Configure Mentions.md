[Mentions](/api-reference/10%20UI%20Components/dxHtmlEditor/1%20Configuration/mentions '/Documentation/ApiReference/UI_Components/dxHtmlEditor/Configuration/mentions/') allow users to reference others within text or conversation threads. This functionality is not used in this tutorial. For information about it, refer to the following demo: [Mentions](https://js.devexpress.com/Demos/WidgetsGallery/Demo/HtmlEditor/Mentions/).
