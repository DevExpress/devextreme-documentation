- **Selection**    
Users can [select](/api-reference/10%20UI%20Components/dxList/1%20Configuration/selectionMode.md '/Documentation/ApiReference/UI_Components/dxList/Configuration/#selectionMode') one or multiple items in the List component.

- **Search**     
Activate the search bar by setting the [searchEnabled](/Documentation/ApiReference/UI_Components/dxList/Configuration/#searchEnabled) property to true.

- **Reorder and Delete Items**    
You can also configure [reordering](/concepts/05%20UI%20Components/List/30%20Item%20Reordering/01%20User%20Interaction.md '/Documentation/Guide/UI_Components/List/Item_Reordering/#User_Interaction') and [deletion](/concepts/05%20UI%20Components/List/35%20Item%20Deletion/01%20User%20Interaction.md '/Documentation/Guide/UI_Components/List/Item_Deletion/#User_Interaction') of List items.

- **Grouping**      
Display items in a two-level hierarchy with [groups](/Documentation/Guide/UI_Components/List/Getting_Started_with_List/#Group_Items). Use the [groupTemplate](/Documentation/ApiReference/UI_Components/dxList/Configuration/#groupTemplate) option to customize group headers.

- **Customize Item Appearance**      
Use the [itemTemplate](/Documentation/ApiReference/UI_Components/dxList/Configuration/#itemTemplate) property to customize the appearance of List items.
