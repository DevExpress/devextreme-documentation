A user can use the following keys to interact with the MultiView component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>&larr; / &uarr;</td>
        <td>Moves focus to the previous view.</td>
    </tr>
    <tr>
        <td>&rarr; / &darr;</td>
        <td>Moves focus to the next view.</td>
    </tr>
    <tr>
        <td>Home</td>
        <td>Moves focus to the first view.</td>
    </tr>
    <tr>
        <td>End</td>
        <td>Moves focus to the last view.</td>
    </tr>
</table>
