- **Data Binding**    
PivotGrid can display data [from various sources](/concepts/05%20UI%20Components/PivotGrid/030%20Data%20Binding/10%20Data%20Binding.md '/Documentation/Guide/UI_Components/PivotGrid/Data_Binding/'): an [in-memory array](/concepts/05%20UI%20Components/PivotGrid/030%20Data%20Binding/10%20Provide%20Data/20%20Using%20the%20Client-Side%20Processing/15%20Using%20an%20ArrayStore.md '/Documentation/Guide/UI_Components/PivotGrid/Data_Binding/#Provide_Data/Using_the_Client-Side_Processing/Using_an_ArrayStore'), [HTML5 web storage](/concepts/05%20UI%20Components/PivotGrid/030%20Data%20Binding/10%20Provide%20Data/20%20Using%20the%20Client-Side%20Processing/20%20Using%20a%20LocalStore.md '/Documentation/Guide/UI_Components/PivotGrid/Data_Binding/#Provide_Data/Using_the_Client-Side_Processing/Using_a_LocalStore'), [OData service](/concepts/05%20UI%20Components/PivotGrid/030%20Data%20Binding/10%20Provide%20Data/20%20Using%20the%20Client-Side%20Processing/30%20Using%20an%20ODataStore.md '/Documentation/Guide/UI_Components/PivotGrid/Data_Binding/#Provide_Data/Using_the_Client-Side_Processing/Using_an_ODataStore'), [OLAP server](/concepts/05%20UI%20Components/PivotGrid/030%20Data%20Binding/10%20Provide%20Data/30%20Using%20an%20OLAP.md '/Documentation/Guide/UI_Components/PivotGrid/Data_Binding/#Provide_Data/Using_an_OLAP'), or a [customized data source](/concepts/05%20UI%20Components/PivotGrid/030%20Data%20Binding/10%20Provide%20Data/20%20Using%20the%20Client-Side%20Processing/40%20Using%20a%20CustomStore.md '/Documentation/Guide/UI_Components/PivotGrid/Data_Binding/#Provide_Data/Using_the_Client-Side_Processing/Using_a_CustomStore'). To process data from the source, use [PivotGridDataSource](/Documentation/ApiReference/Data_Layer/PivotGridDataSource/).

- **Summary Calculation Modes**    
PivotGrid displays [value summaries](/concepts/05%20UI%20Components/PivotGrid/040%20Summaries '/Documentation/Guide/UI_Components/PivotGrid/Summaries/') (totals) in its cells. Choose between [predefined summaries](/concepts/05%20UI%20Components/PivotGrid/040%20Summaries/20%20Predefined%20Summaries.md '/Documentation/Guide/UI_Components/PivotGrid/Summaries/#Predefined_Summaries'), [aggregate](/concepts/05%20UI%20Components/PivotGrid/040%20Summaries/07%20Custom%20Aggregate%20Function '/Documentation/Guide/UI_Components/PivotGrid/Summaries/#Custom_Aggregate_Function') values, or a implement a [custom summary](/concepts/05%20UI%20Components/PivotGrid/040%20Summaries/30%20Custom%20Summaries.md '/Documentation/Guide/UI_Components/PivotGrid/Summaries/#Custom_Summaries'). PivotGrid can also calculate [running totals](/Documentation/ApiReference/Data_Layer/PivotGridDataSource/Configuration/fields/#runningTotal) by rows or columns.

- **Customizable Layout**    
PivotGrid allows you to [customize its layout](https://js.devexpress.com/Demos/WidgetsGallery/Demo/PivotGrid/LayoutCustomization/). You can enable or disable multiple PivotGrid elements. Review the [Visual Elements](/concepts/05%20UI%20Components/PivotGrid/010%20Visual%20Elements '/Documentation/Guide/UI_Components/PivotGrid/Visual_Elements/') article for more information.

- **Grouping, Sorting, and Filtering**    
PivotGrid [groups data](/concepts/05%20UI%20Components/PivotGrid/050%20Grouping '/Documentation/Guide/UI_Components/PivotGrid/Grouping/') for header items and fields. [Sort](/concepts/05%20UI%20Components/PivotGrid/060%20Sorting/10%20Sorting.md '/Documentation/Guide/UI_Components/PivotGrid/Sorting/') or [filter](/concepts/05%20UI%20Components/PivotGrid/080%20Filtering '/Documentation/Guide/UI_Components/PivotGrid/Filtering/') fields through the UI or programmatically.

- **Export to Excel**    
[Export PivotGrid data to Excel](/concepts/05%20UI%20Components/PivotGrid/090%20Export/00%20Export.md '/Documentation/Guide/UI_Components/PivotGrid/Export/') in XLSX or CSV format and customize the exported cells.

- **Chart Integration**    
PivotGrid handles large numeric datasets that can be hard to interpret. For a detailed data visualization, [use the Chart component](/concepts/05%20UI%20Components/PivotGrid/150%20Integration%20with%20Chart/10%20Integration%20with%20Chart.md '/Documentation/Guide/UI_Components/PivotGrid/Integration_with_Chart/') that integrates into PivotGrid.

- **Virtual Scrolling**    
If PivotGrid is bound to a large dataset, you can [enable a virtual scrolling mode](/Documentation/ApiReference/UI_Components/dxPivotGrid/Configuration/scrolling/#mode) to optimize data load time and reduce resource consumption.

- **Drill Down**    
PivotGrid supports [drill-down operations](https://js.devexpress.com/Demos/WidgetsGallery/Demo/PivotGrid/DrillDown/) that allow you to retrieve individual records for calculating a specific summary value.
