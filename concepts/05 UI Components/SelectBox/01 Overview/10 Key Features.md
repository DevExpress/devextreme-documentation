- **Selection**    
Users can select items from a drop-down list. You can [handle the selection](/Documentation/ApiReference/UI_Components/dxSelectBox/Configuration/#onSelectionChanged).

- **Custom Values**    
Allow users to [add new items](/Documentation/Guide/UI_Components/SelectBox/Create_a_User-Defined_Item/) to SelectBox in addition to selecting from the existing items.

- **Display Groups**    
[Group](/Documentation/Guide/UI_Components/SelectBox/Grouping/In_the_Data_Source/) SelectBox items for better readability. Only one-level grouping is supported.

- **Input-Based Search**    
Configure [search settings](/Documentation/Guide/UI_Components/SelectBox/Configure_Search_Parameters/) based on user input.
