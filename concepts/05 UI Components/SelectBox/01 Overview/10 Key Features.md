- **Selection**    
Users can select items from a drop-down list. You can [handle the selection](/api-reference/10%20UI%20Components/dxSelectBox/1%20Configuration/onSelectionChanged.md '/Documentation/ApiReference/UI_Components/dxSelectBox/Configuration/#onSelectionChanged').

- **Custom Values**    
Allow users to [add new items](/concepts/05%20UI%20Components/SelectBox/15%20Create%20a%20User-Defined%20Item.md '/Documentation/Guide/UI_Components/SelectBox/Create_a_User-Defined_Item/') to SelectBox in addition to selecting from the existing items.

- **Display Groups**    
[Group](/concepts/05%20UI%20Components/SelectBox/07%20Grouping/01%20In%20the%20Data%20Source.md '/Documentation/Guide/UI_Components/SelectBox/Grouping/In_the_Data_Source/') SelectBox items for better readability. Only one-level grouping is supported.

- **Input-Based Search**    
Configure [search settings](/concepts/05%20UI%20Components/SelectBox/10%20Configure%20Search%20Parameters.md '/Documentation/Guide/UI_Components/SelectBox/Configure_Search_Parameters/') based on user input.
