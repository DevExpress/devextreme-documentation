A user can use the following keys to interact with the SelectBox component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>Alt + Down Arrow</td>
        <td>Opens the drop-down list.</td>
    </tr>
    <tr>
        <td>Alt + Up Arrow</td>
        <td>Closes the drop-down list.</td>
    </tr>
    <tr>
        <td>Esc</td>
        <td>Closes the drop-down list and removes focus from the UI component.</td>
    </tr>
    <tr>
        <td>Up/Down Arrows</td>
        <td>If the drop-down list is opened, moves focus to the previous/next item; otherwise, selects the previous/next item.</td>
    </tr>
    <tr>
        <td>Page Up / Page Down</td>
        <td>Moves focus to the first/last item.</td>
    </tr>
    <tr>
        <td>Enter or Space</td>
        <td>Selects the item.</td>
    </tr>
    <tr>
        <td>Backspace or Delete</td>
        <td>Clears selection if <a href="/Documentation/ApiReference/UI_Components/dxSelectBox/Configuration/#showClearButton">showClearButton</a> is enabled.</td>
    </tr>
</table>
