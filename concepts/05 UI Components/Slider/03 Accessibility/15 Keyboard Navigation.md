A user can use the following keys to interact with the Slider component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>&larr; / &rarr; </td>
        <td>Decreases/increases the value of the focused handle by a <a href="/Documentation/ApiReference/UI_Components/dxSlider/Configuration/#step">step</a>.</td>
    </tr>
    <tr>
        <td>Page Up / Page Down</td>
        <td>Decreases/increases the value of the focused handle by a <a href="/Documentation/ApiReference/UI_Components/dxSlider/Configuration/#keyStep">keyStep</a>.</td>
    </tr>
    <tr>
        <td>Home / End</td>
        <td>Sets the focused handle to the <a href="/Documentation/ApiReference/UI_Components/dxSlider/Configuration/#min">minimum</a>/<a href="/Documentation/ApiReference/UI_Components/dxSlider/Configuration/#max">maximum</a> value or to the value of the other handle.</td>
    </tr>
</table>
