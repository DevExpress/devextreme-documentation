- **Multiple Progression Options**    
Users navigate Stepper sequentially when the [linear](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#linear) property is set to `true`. You can allow users to skip steps by setting **linear** to `false`. Switch between left-to-right and right-to-left progression with the [rtlEnabled](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#rtlEnabled) property. 
- **Customizable Appearance**    
The Stepper component offers multiple options like customizable [orientation](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#orientation), step indicators, [labels](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#label), and [hints](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#hint). You can customize Stepper further with item templates by defining [itemTemplate](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#itemTemplate) or [items[]](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/).[template](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#template).
- **Selection Options**    
You can get and set the selected step with [selectedItem](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#selectedItem) and [selectedIndex](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#selectedIndex). Implement custom logic upon selection change with [onSelectionChanged](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#onSelectionChanged) and [onSelectionChanging](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#onSelectionChanging). Stepper allows users to change selection with the arrow keys when [selectOnFocus](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#selectOnFocus) is set to `true`.
