- **Multiple Progression Options**    
Users navigate Stepper sequentially when the [linear](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/linear.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#linear') property is set to `true`. You can allow users to skip steps by setting **linear** to `false`. Switch between left-to-right and right-to-left progression with the [rtlEnabled](/api-reference/10%20UI%20Components/DOMComponent/1%20Configuration/rtlEnabled.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#rtlEnabled') property. 
- **Customizable Appearance**    
The Stepper component offers multiple options like customizable [orientation](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/orientation.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#orientation'), step indicators, [labels](/api-reference/10%20UI%20Components/dxStepper/7%20Interfaces/dxStepperItem/label.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#label'), and [hints](/api-reference/10%20UI%20Components/dxStepper/7%20Interfaces/dxStepperItem/hint.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#hint'). You can customize Stepper further with item templates by defining [itemTemplate](/api-reference/10%20UI%20Components/CollectionWidget/1%20Configuration/itemTemplate.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#itemTemplate') or [items[]](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/).[template](/api-reference/_hidden/CollectionWidgetItem/template.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#template').
- **Selection Options**    
You can get and set the selected step with [selectedItem](/api-reference/10%20UI%20Components/CollectionWidget/1%20Configuration/selectedItem.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#selectedItem') and [selectedIndex](/api-reference/10%20UI%20Components/CollectionWidget/1%20Configuration/selectedIndex.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#selectedIndex'). Implement custom logic upon selection change with [onSelectionChanged](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/onSelectionChanged.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#onSelectionChanged') and [onSelectionChanging](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/onSelectionChanging.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#onSelectionChanging'). Stepper allows users to change selection with the arrow keys when [selectOnFocus](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/selectOnFocus.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#selectOnFocus') is set to `true`.
