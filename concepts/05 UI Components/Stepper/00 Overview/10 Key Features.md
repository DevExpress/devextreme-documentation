- **Multiple Progression Options**    
Switch between left-to-right and right-to-left progression with the [rtlEnabled]() property. Toggle between sequential and out-of-order progression with the [linear]() property.
- **Customizable Appearance**    
The Stepper component offers multiple options like customizable [orientation](), step indicators, [labels](), and [hints](). You can customize Stepper further with item templates by defining itemTemplate or items[].[itemTemplate]().
- **Validation**    
Stepper integrates with the DevExtreme [validation engine](/concepts/05%20UI%20Components/Form/01%20Getting%20Started%20with%20Form/70%20Validate%20the%20Form%20Data.md '/Documentation/Guide/UI_Components/Form/Getting_Started_with_Form/#Validate_the_Form_Data'). Set [isValid]() to false to indicate validation errors.
- **Keyboard Navigation**    
Stepper supports keyboard navigation. Users can switch between steps with the arrow keys.
