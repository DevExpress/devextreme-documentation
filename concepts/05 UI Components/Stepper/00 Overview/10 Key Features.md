- **Orientation Options**    
Stepper supports horizontal and vertical [orientation](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/orientation.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#orientation') options.
- **Navigation Modes**    
The [linear](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/linear.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#linear') property enables sequential navigation. Disable **linear** to allow users to skip steps.
- **Step Types and States**    
The component offers various step types and states such as [optional](/api-reference/10%20UI%20Components/dxStepper/7%20Interfaces/dxStepperItem/optional.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#optional') and [disabled](/api-reference/_hidden/CollectionWidgetItem/disabled.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#disabled'). You can also indicate step validity with the [isValid](/api-reference/10%20UI%20Components/dxStepper/7%20Interfaces/dxStepperItem/isValid.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#isValid') property.

- **Step Customization**    
Stepper offers multiple step customization options like configurable step [labels](/api-reference/10%20UI%20Components/dxStepper/7%20Interfaces/dxStepperItem/label.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#label') and [hints](/api-reference/10%20UI%20Components/dxStepper/7%20Interfaces/dxStepperItem/hint.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#hint'). You can also set custom step [icons](/api-reference/10%20UI%20Components/dxStepper/7%20Interfaces/dxStepperItem/icon.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#icon') or [text](/api-reference/_hidden/CollectionWidgetItem/text.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#text').
- **Advanced Customization with Templates**    
Implement in-depth customizations such as custom step shapes with item templates. To do so, define [itemTemplate](/api-reference/10%20UI%20Components/CollectionWidget/1%20Configuration/itemTemplate.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#itemTemplate') or [items[]](/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/).[template](/api-reference/_hidden/CollectionWidgetItem/template.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/items/#template').

- **Selection Options**    
You can get and set the selected step with [selectedItem](/api-reference/10%20UI%20Components/CollectionWidget/1%20Configuration/selectedItem.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#selectedItem') and [selectedIndex](/api-reference/10%20UI%20Components/CollectionWidget/1%20Configuration/selectedIndex.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#selectedIndex'). Implement custom logic upon selection change with [onSelectionChanged](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/onSelectionChanged.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#onSelectionChanged') and [onSelectionChanging](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/onSelectionChanging.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#onSelectionChanging').
- **Keyboard Navigation**    
Stepper supports [keyboard navigation](/concepts/05%20UI%20Components/Stepper/10%20Accessibility/15%20Keyboard%20Navigation.md '/Documentation/Guide/UI_Components/Stepper/Accessibility/#Keyboard_Navigation'). The component automatically selects steps focused with the keyboard if [selectOnFocus](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/selectOnFocus.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#selectOnFocus') is enabled.
- **Right-to-Left Mode**    
Switch between left-to-right and right-to-left progression on horizontal Steppers with the [rtlEnabled](/api-reference/10%20UI%20Components/DOMComponent/1%20Configuration/rtlEnabled.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#rtlEnabled') property. **rtlEnabled** changes the placement of step labels on vertical Steppers.
