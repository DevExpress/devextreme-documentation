Specify the following settings to configure Stepper:

- [linear](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/linear.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#linear'): Specifies whether users can skip steps.
- [orientation](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/orientation.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#orientation'): Sets the orientation of Stepper.
- [selectOnFocus](/api-reference/10%20UI%20Components/dxStepper/1%20Configuration/selectOnFocus.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#selectOnFocus'): Specifies whether steps focused with keyboard arrows are selected automatically.
- [rtlEnabled](/api-reference/10%20UI%20Components/DOMComponent/1%20Configuration/rtlEnabled.md '/Documentation/ApiReference/UI_Components/dxStepper/Configuration/#rtlEnabled'): Switches the Stpper to Right-to-Left mode.
