A user can use the following keys to interact with the TabPanel component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>&larr;</td>
        <td>Switches the UI component to the previous view.</td>
    </tr>
    <tr>
        <td>&rarr;</td>
        <td>Switches the UI component to the next view.</td>
    </tr>
</table>
