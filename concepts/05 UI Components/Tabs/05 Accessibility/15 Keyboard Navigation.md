A user can use the following keys to interact with the Tabs component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>Left/Up Arrows, PageUp</td>
        <td>Moves focus to the previous tab.</td>
    </tr>
    <tr>
        <td>Right/Down Arrows, PageDown</td>
        <td> Moves focus to the next tab.</td>
    </tr>
    <tr>
        <td>Home</td>
        <td>Moves focus to the first tab.</td>
    </tr>
    <tr>
        <td>End</td>
        <td>Moves focus to the last tab.</td>
    </tr>
    <tr>
        <td>Enter<br>Space</td>
        <td>Selects the focused tab.</td>
    </tr>
    <tr>
        <td>Shift + Mouse Wheel</td>
        <td>Scrolls the tabs left/right if the <a href="/Documentation/ApiReference/UI_Components/dxTabs/Configuration/#scrollingEnabled">scrollingEnabled</a> property is <b>true</b>.</td>
    </tr>
</table>
