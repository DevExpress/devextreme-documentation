A user can use the following keys to interact with the Toolbar component: 

<table class="dx-table">
    <tr>
        <th>Key</th>
        <th>Action</th>
    </tr>
    <tr>
        <td>Tab</td>
        <td>Moves focus to the next toolbar item.</td>
    </tr>
    <tr>
        <td>Shift + Tab</td>
        <td>Moves focus to the previous toolbar item.</td>
    </tr>
</table>
