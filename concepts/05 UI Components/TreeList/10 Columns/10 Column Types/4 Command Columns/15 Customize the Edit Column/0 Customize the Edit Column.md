The edit column appears when [editing](/api-reference/10%20UI%20Components/dxTreeList/1%20Configuration/editing '/Documentation/ApiReference/UI_Components/dxTreeList/Configuration/editing/') is allowed. The avaiable editing buttons depend on the [editing mode](/api-reference/10%20UI%20Components/dxTreeList/1%20Configuration/editing '/Documentation/ApiReference/UI_Components/dxTreeList/Configuration/editing/'#mode).

The edit column is a type of buttons column. To customize it, set the column's [type](/api-reference/_hidden/dxTreeListColumn/type.md '/api-reference/10%20UI%20Components/dxTreeList/1%20Configuration/columns '/Documentation/ApiReference/UI_Components/dxTreeList/Configuration/columns/'#type') to *"buttons"* and specify the [other properties](/api-reference/10%20UI%20Components/dxTreeList/1%20Configuration/columns '/Documentation/ApiReference/UI_Components/dxTreeList/Configuration/columns/').
