Drag & drop allows users to reorder nodes and change their hierarchy. This functionality is not used in this tutorial. For information about it, refer to the following demos: [Drag & Drop for Plain Data Structure](https://js.devexpress.com/Demos/WidgetsGallery/Demo/TreeView/DragAndDropPlainDataStructure/) and [Drag & Drop for Hierarchical Data Structure](https://js.devexpress.com/Demos/WidgetsGallery/Demo/TreeView/DragAndDropHierarchicalDataStructure/).
