Users can drag and drop nodes to rearrange them and change the data hierarchy. This functionality is not used in this tutorial. For details, refer to the following demos: [Drag & Drop for Plain Data Structure](https://js.devexpress.com/Demos/WidgetsGallery/Demo/TreeView/DragAndDropPlainDataStructure/) and [Drag & Drop for Hierarchical Data Structure](https://js.devexpress.com/Demos/WidgetsGallery/Demo/TreeView/DragAndDropHierarchicalDataStructure/).
