- **Data Binding**    
TreeView can load and update data from different [data source](/api-reference/10%20UI%20Components/dxTreeView/1%20Configuration/dataSource.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#dataSource') types. You can define data as [plain or hierarchical](/api-reference/10%20UI%20Components/dxTreeView/1%20Configuration/dataStructure.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#dataStructure'). If you choose plain data, you can load it in a [virtual mode](/concepts/05%20UI%20Components/TreeView/32%20Enhance%20Performance%20on%20Large%20Datasets.md '/Documentation/Guide/UI_Components/TreeView/Enhance_Performance_on_Large_Datasets/') to improve performance with large datasets.

- **Search**    
You can enable [search](/concepts/05%20UI%20Components/TreeView/15%20Search%20Nodes.md '/Documentation/Guide/UI_Components/TreeView/Search_Nodes/') in the TreeView component, allowing node searches if they are loaded.

- **Selection**     
When selection is enabled, TreeView [displays check boxes](/api-reference/10%20UI%20Components/dxTreeView/1%20Configuration/showCheckBoxesMode.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#showCheckBoxesMode'). You can specify various [selection modes](/api-reference/10%20UI%20Components/dxTreeView/1%20Configuration/selectionMode.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#selectionMode'). To handle selection, use event handlers such as [onSelectionChanged](/api-reference/10%20UI%20Components/dxTreeView/1%20Configuration/onSelectionChanged.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#onSelectionChanged'), [onSelectAllValueChanged](/api-reference/10%20UI%20Components/dxTreeView/1%20Configuration/onSelectAllValueChanged.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#onSelectAllValueChanged'), and [onItemSelectionChanged](/api-reference/10%20UI%20Components/dxTreeView/1%20Configuration/onItemSelectionChanged.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#onItemSelectionChanged').

- **Customization**     
You can customize TreeView appearance. Set custom [collapse](/api-reference/10%20UI%20Components/dxTreeView/1%20Configuration/collapseIcon.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#collapseIcon') and [expand](/api-reference/10%20UI%20Components/dxTreeView/1%20Configuration/expandIcon.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#expandIcon') icons. Specify a custom template for [all items](/api-reference/10%20UI%20Components/dxTreeView/1%20Configuration/itemTemplate.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#itemTemplate') or [individually](/api-reference/_hidden/CollectionWidgetItem/template.md '/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/items/#template') for each item.
