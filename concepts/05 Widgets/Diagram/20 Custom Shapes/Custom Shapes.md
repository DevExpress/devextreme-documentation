The **Diagram** UI component provides a collection of built-in shapes. Use the [customShapes](/api-reference/10%20UI%20Widgets/dxDiagram/1%20Configuration/customShapes '/Documentation/ApiReference/UI_Widgets/dxDiagram/Configuration/customShapes/') property to extend this collection with custom shapes.

The [toolbox](/api-reference/10%20UI%20Widgets/dxDiagram/1%20Configuration/toolbox '/Documentation/ApiReference/UI_Widgets/dxDiagram/Configuration/toolbox/') property allows you to add the custom shapes to the toolbox.

A custom shape can be created based on a default shape type or with a custom background image. The [type](/api-reference/10%20UI%20Widgets/dxDiagram/1%20Configuration/customShapes/type.md '/Documentation/ApiReference/UI_Widgets/dxDiagram/Configuration/customShapes/#type') property identifies custom shapes and should be unique.
