To set a required color for the background, use the [color](/api-reference/20%20Data%20Visualization%20Widgets/dxRangeSelector/1%20Configuration/background/color.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxRangeSelector/Configuration/background/#color') option within the **background** configuration object.

	<!--JavaScript-->
	var rangeSelectorOptions = {
		// ...
		background: {
			color: '#FFE4B5'
		}
	};
