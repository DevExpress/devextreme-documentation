This guide reviews frequent DevExtreme React component issues. The issues can be the following:

- The component works incorrectly on click. 
- The component discards its value on input.
- The component displays data incorrectly (for example, the [DataGrid]() shows gray placeholders).
- The element selection does not work.
- The render or function component work unexpectedly.
- Slow performance.

You can follow the steps below to analyze whether an unnecessary component re-render causes your issue.
