DevExtreme includes an icon library with SVG and font icons for all DevExtreme themes. You can use an icon [in UI components](/concepts/60%20Themes%20and%20Styles/30%20Icons/20%20Icons%20in%20Widgets.md '/Documentation/Guide/Themes_and_Styles/Icons/#Icons_in_DevExtreme_UI_Components') and [in other page elements](/concepts/60%20Themes%20and%20Styles/30%20Icons/30%20Icons%20in%20Other%20HTML%20Elements.md '/Documentation/Guide/Themes_and_Styles/Icons/#Icons_in_Other_HTML_Elements') as is or [customize it](/concepts/60%20Themes%20and%20Styles/30%20Icons/35%20Customize%20Icons.md '/Documentation/Guide/Themes_and_Styles/Icons/#Customize_Icons').
