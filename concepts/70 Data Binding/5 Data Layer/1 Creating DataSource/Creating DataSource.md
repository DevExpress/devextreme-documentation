To create a [DataSource](/api-reference/30%20Data%20Layer/DataSource '/Documentation/ApiReference/Data_Layer/DataSource/') instance, call the DataSource constructor and pass the required [configuration object](/api-reference/30%20Data%20Layer/DataSource/1%20Configuration '/Documentation/ApiReference/Data_Layer/DataSource/Configuration/') to this constructor. This section covers the ways to associate the required data with the DataSource instance being created. You can use one of these options to create a DataSource instance.
