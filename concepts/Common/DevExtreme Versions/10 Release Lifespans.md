Each major DevExtreme release is supported for the following durations:

<table class="dx-table">
    <tr>
        <th>Update Type</th>
        <th>Duration</th>
    </tr>
    <tr>
        <td>Minor/Maintenance Updates</td>
        <td>One Year</td>
    </tr>
    <tr>
        <td>Security-Related Updates</td>
        <td>Two Years</td>
    </tr>
</table>

Major release lifespans intersect. For instance, v25.1.8 was released at a later date than v25.2.3. However, v25.2.3 is the later version in terms of feature availability.

For more information, refer to the following topic: [Supported Versions, Version History, and IDE Prerequisites](https://www.devexpress.com/support/versions.xml).
