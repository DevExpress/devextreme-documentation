The DevExpress MCP server supports the following IDEs and AI agents:

**IDEs**:

- Visual Studio 2022 (version 17.14 or later)
- Visual Studio Code
- Cursor IDE
- JetBrains Rider 2025.1+

**AI agents**:

- GitHub Copilot
- JetBrains AI Assistant
- Claude/Claude Code
- Other MCP-compatible AI Assistants

[note] We strongly recommend you keep your development tools up to date to ensure access to the latest MCP features.
