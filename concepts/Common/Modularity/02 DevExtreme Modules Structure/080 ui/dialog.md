Exposes an object that serves as a namespace for methods displaying a message in an application/site.

Exports:

- *alert* - The [alert](/Documentation/ApiReference/Common/utils/ui/dialog/#alertmessage_title) method

        require("ui/dialog").alert;
        // ===== or =====
        import { alert } from "devextreme/ui/dialog";

- *confirm* - The [confirm](/Documentation/ApiReference/Common/utils/ui/dialog/#confirmmessage_title) method

        require("ui/dialog").confirm;
        // ===== or =====
        import { confirm } from "devextreme/ui/dialog";
