[DevExtreme Dialog UI methods](/Documentation/ApiReference/Common/Utils/ui/dialog/#alertmessageHtml_title) accept an HTML string as a dialog message. This string is not encoded. You can use the `encodeHtml` utility method to encode the HTML string before it is passed to a Dialog UI method. The following example illustrates this technique: <a href="https://codepen.io/romantsukanov/pen/BazxOaE?editors=1010" target="_blank">HTML Encoding in a Dialog UI Method</a>.
