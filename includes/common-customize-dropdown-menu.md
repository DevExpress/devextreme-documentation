The drop-down menu is the [Popup](/concepts/05%20Widgets/Popup/00%20Overview.md '/Documentation/Guide/UI_Components/Popup/Overview/') UI component.

To customize the Popup, use the [dropDownOptions](/api-reference/10%20UI%20Widgets/dx{WidgetName}/1%20Configuration/dropDownOptions.md '/Documentation/ApiReference/UI_Components/dx{WidgetName}/Configuration/#dropDownOptions') object.
