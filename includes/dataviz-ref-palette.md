This option accepts either the name of a [predefined palette](/concepts/60%20Themes%20and%20Styles/20%20SVG-Based%20Widgets%20Customization/10%20Palettes/00%20Palettes.md '/Documentation/Guide/Themes_and_Styles/SVG-Based_Widgets_Customization/#Palettes') or an array of colors. The array can include the following colors:

* Hexadecimal colors
* RGB colors
* RGBA colors
* Predefined/cross-browser color names
* <a href="http://www.w3.org/TR/SVG/types.html#ColorKeywords" target="_blank">Predefined SVG colors</a>
