All the settings in this section are specified as follows:

- The [commonSeriesSettings]({basewidgetpath}/Configuration/commonSeriesSettings/) object     
Specifies settings for all series in a **{WidgetName}**.

- The **commonSeriesSettings**.[${{seriesType}}]({basewidgetpath}/Configuration/commonSeriesSettings/#${{seriesType}}) object        
Specifies settings for all series of the **${{seriesType}}** type.

- An object in the [series]({basewidgetpath}/Configuration/series/) array        
Specifies settings for an individual series.
