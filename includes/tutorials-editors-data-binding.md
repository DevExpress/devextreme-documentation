The {ComponentName} can load data from different data source types. To use a local array, assign it to the [dataSource](/Documentation/ApiReference/UI_Components/dx{ComponentName}/Configuration/#dataSource) property. If array elements are objects, set the fields that supply the {ComponentName}'s value ([valueExpr](/Documentation/ApiReference/UI_Components/dx{ComponentName}/Configuration/#valueExpr)) and displayed value ([displayExpr](/Documentation/ApiReference/UI_Components/dx{ComponentName}/Configuration/#displayExpr)). - [OData](/Documentation/Guide/Data_Binding/Specify_a_Data_Source/OData/)
