This option accepts one of the following values.

- A [CSS selector](https://developer.mozilla.org/en-US/docs/Learn/CSS/Introduction_to_CSS/Selectors), or a [jQuery selector](https://api.jquery.com/category/selectors/) if you use jQuery

        target: '#targetElement';

- A jQuery wrapper

        target: $('#targetElement');

- A DOM element

        target: document.getElementById('#targetElement');
